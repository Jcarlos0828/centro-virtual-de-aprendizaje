Para realizar este curso, el participante necesita tener acceso a un equipo de cómputo que cumpla con las siguientes especificaciones de hardware y software:
Hardware

• Una computadora tipo PC con acceso a Internet.
• Procesador 486 o superior.
• Memoria RAM de 64MB o superior.
• Controlador de apuntador (ratón, almohadilla, puntero táctil, etc.)
• Impresora de texto y gráfica.

Software

• Sistema operativo Windows
• Navegador de Internet (se recomienda Internet explorer)
