El participante desempeña un papel clave en el proceso de aprendizaje a distancia, éste debe comprometerse consigo mismo a:

Tener una actitud positiva para estudiar en una modalidad a distancia para que su aprovechamiento sea el óptimo.

Participar activamente para aprender de forma autónoma, es decir, por si mismo.

Establecer su propio horario y ritmo de estudio para completar el curso.

Acceder periódicamente el curso; dedicarse a conocerlo; navegar y hacer una revisión de la información, acudir a los diferentes apartados para conocer el temario, la metodología, las políticas del curso; revisar y analizar los contenidos; realizar las actividades y ejercicios.

Disponer de un tiempo continuo para estudiar o realizar ininterrumpidamente un tema o una actividad completa.

Concluir las actividades del curso dentro del período acordado.
