Cómo se transmite el virus del SIDA.

El SIDA tiene sólo tres formas de contagio; es decir, sólo podría enfermarse de SIDA si usted:

  • Tiene relaciones sexuales con una persona infectada con el virus. Esta forma de transmisión se llama: sexual.
  • Recibe en una transfusión la sangre de una persona infectada con el virus o comparte jeringas, agujas o instrumentos que estén contaminados con el virus. Esta forma de transmisión se llama: sanguínea.
  • La tercera forma de adquirir la enfermedad es durante el embarazo o el nacimiento de un niño. Si la madre está infectada con el virus del SIDA, puede transmitírselo a su hijo. Esta forma de transmisión se llama: perinatal.
