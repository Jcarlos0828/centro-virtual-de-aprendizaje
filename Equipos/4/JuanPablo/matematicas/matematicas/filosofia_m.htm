Filosofía educativa

  1. La metodología usada está basada en el Estudio de Casos.

  2. En la presentación de la mayoría de los casos vienen unas tablas interactivas. Éstas son muy importantes para el proceso de enseñanza-aprendizaje, y facilitan mucho la comprensión y el análisis de los diferentes modelos matemáticos. En las tablas interactivas se pueden modificar los parámetros del problema y al mismo tiempo observar, en una gráfica y en una tabla, los cambios que estas modificaciones generan en el resultado.

  3. Dado un caso o problema, el procedimiento para resolverlo es el siguiente:

    3.1 Se analiza la situación para construir un modelo matemático para el caso.
    3.2 Se establece la representación algebraica del modelo como una ecuación o un sistema de ecuaciones de diferencia finita.
    3.3 Se resuelve el problema utilizando la representación algebraica del modelo. En caso de que la solución total sea muy larga, se resuelve para uno o dos períodos, y después se termina de resolver en Excel.
    3.4 En los casos en que se pueda tener una fórmula, se hace la deducción de ésta utilizando la representación algebraica del modelo.
    3.5 Cuando es posible, se utiliza la fórmula para llegar a la solución del caso.
    3.6 Se plantea el problema en Excel para resolverlo. El planteamiento del problema en Excel puede hacerse utilizando el modelo o la representación algebraica del modelo, o bien utilizando la fórmula directamente.
    3.7 Se hace un análisis de los resultados obtenidos.
