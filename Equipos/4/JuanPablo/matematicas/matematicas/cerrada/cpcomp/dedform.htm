Deducción de la fórmula de
Crecimiento Poblacional Compuesto

Para hacer la deducción de la fórmula, se va utilizar la representación algebraica para un modelo de Crecimiento Poblacional Compuesto        Pt = Pt-1( 1+ r )

La población después de un año, de acuerdo a la representación algebraica del modelo, sería:
P1= P0 ( 1 + r )
La población después de un año es…………………………… P1= P0 ( 1 + r )

La población después de dos años de acuerdo a la representación algebraica del modelo, sería.
P2 = P1( 1 + r )            sustituyendo el valor de P1 obtenido en la ecuación anterior se tiene
P2 = P0 ( 1 + r )( 1 + r ) o bien,
P2 = P0 ( 1 + r )2
La población después de 2 años es …………………………….. P2 = P0 ( 1 + r )2

Tomando la representación algebraica del modelo para tres años, la población sería:
P3 = P2( 1 + r )         sustituyendo el valor de la población después de 2 años se tiene que
P3 = P0 ( 1 + r )2( 1 + r )     o bien,
P3 = P0 ( 1 + r )3
La población después de 3 años es ……………………………. P3 = P0 ( 1 + r )3

Se puede observar que el exponente del factor ( 1 + r ) es igual al número de años transcurridos después del año inicial. Así el exponente 3 corresponde a la población después de tres años, el 2 a la población después de dos años.
Siguiendo este procedimiento podemos deducir de acuerdo a la observación que la población después de t años es Pt = P0( 1 + r )t.
La población después de t años es………………………………Pt = P0( 1 + r )t

 

La fórmula para encontrar la población después de t años en un Crecimiento Poblacional Compuesto es:

Pt = P0( 1 + r )t

en donde,

Pt     es la población total después de t años.
P0    es la población inicial.
r      es la tasa de crecimiento anual expresada en decimales.
t      es el número de años después del año inicial.
