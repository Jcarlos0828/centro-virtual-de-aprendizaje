Caso 3
Número de habitantes de un poblado en donde la población está decreciendo
Solución utilizando la fórmula de Crecimiento Poblacional Compuesto

El número de habitantes de un poblado era 80,000 en 1998. Por emigración a la ciudad, la población decrece a razón de 0.8% anual . Encuentre el número de habitantes para el año 2000.

Al utilizar la fórmula de Crecimiento Poblacional Compuesto Pt = P0 (1 + r)t   en una disminución de población, hay que tener cuidado de tomar la tasa de crecimiento como negativa (- 0.008).

Datos:
P0 = 80,000
r =  -0.008
t = 2 años
P2 = ? población total después de dos años

Solución:
Aplicando la fórmula para dos años, queda la expresión:
P2 = P0 (1 + r)2 sustituyendo los valores de P0 y r se tiene,
P2 = 80,000 (1 – 0.008)2
P2 = 80,000 (0.992)2
P2 = 80,000 (0.984064)
P2 = 78,725.12
El número de habitantes después de dos años es de 78, 725, o bien,
El número de habitantes estimado de habitantes para el 2000 es de 78,725.
