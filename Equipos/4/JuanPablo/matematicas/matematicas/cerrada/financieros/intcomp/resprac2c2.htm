Práctica de autoevaluación 7
  1. El saldo para el primer semestre va a ser igual al saldo inicial (30,000.00) más los intereses. La tasa de interés para cada semestre es igual a la tasa de interés annual dividido entre 2 semestres que tiene el año.
    tasa de interés semestral = 36 / 2 = 18%.
    Los intereses para el primer semestre son el saldo inicial (30,000.00) por la tasa de interés semestral en su forma decimal (0.18)
    intereses del primer semestre = 30,000.00 x 0.18
    intereses del primer semestre = 5,400.00 pesos.
    saldo del primer semestre = Saldo Inicial + Intereses
    saldo del primer semestre = 30,000.00 + 5,400.00
    saldo del primer semestre = 35,400.00
    Para los siguientes semestres se van a utilizar los elementos definidos:

    S2 = S1 + S1 x i = 35,400.00 + 35,400.00 x 0.18 = 35,400.00 + 6,372.00 = 41,772.00
    S3 = S2 + S2 x i = 41,772.00 + 41,772.00 x 0.18 = 41,772.00 + 7,518.96 = 49,290.96
    S4 = S3 + S3 x i = 49,290.96 + 49,290.96 x 0.18 = 49,290.96 + 8,872.37 = 58,163.33
    Al vencer el documento tiene que pagar $58,163.33 pesos.

 

  1. Total de intereses = Saldo Final – Saldo Inicial = 58,163.33 - 30,000.00 = 28,163.33
    El total de intereses que se va a pagar es de $28,163.33 pesos.
