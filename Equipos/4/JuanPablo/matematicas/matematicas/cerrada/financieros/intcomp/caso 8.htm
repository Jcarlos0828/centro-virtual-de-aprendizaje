Caso 8: Pago de un préstamo a 4 años con tasa de interés compuesto anualizado, capitalizable cada semestre y con pagos semestrales.

 

Alicia Cantú, propietaria de una tienda de cerámica, le prestó $35,000.00 a su hermano Luis para ayudarlo a poner un negocio. Luis piensa liquidarle el préstamo en cuatro años con una tasa de interés anual del 25% capitalizable cada semestre. Además Luis le va a pagar $6,000.00 cada semestre.

¿Qué cantidad le va a deber Luis a Alicia al final de los 4 años?

- Solución utilizando el modelo matemático

Para calcular la tasa de interés para cada semestre se tiene que dividir la tasa de interés anual entre dos, ya que un año tiene 2 semestres.
tasa de interés semestral = 25 / 2
tasa de interés semestral = 12.5%.

El saldo para el primer semestre va a ser igual a el saldo inicial ($35,000.00) más los intereses del semestre.
interés del primer período = 35,000.00 x 0.125
interés del primer período = 4,375.00 pesos.
saldo del primer período sin el pago = saldo anterior + intereses
saldo del primer período sin el pago = 35,000.00 + 4,375.00
saldo del primer período sin el pago = 39,375.00 pesos.
Ahora hay que restarle el pago que va a hacer Luis cada semestre de $6,000.00.
saldo del primer período = 39,375.00 - 6,000.00
saldo del primer período = 33,375.00 pesos.

En el modelo que se definió se tiene que el saldo del primer período va a ser igual al saldo del periodo anterior mas los intereses:
S1 = S0 + S0 x i
En este caso se va a estar haciendo un pago cada período, se tiene que restar el pago al saldo anterior y los intereses, quedando el modelo de la siguiente manera:
S1 = S0 + S0 x i - P

Se va a utilizar el modelo definido para cada uno de los periodos::

S1 = S0 + S0 x i - P = 35,000.00 + 35,000.00 * 0.125 - 6,000.00 = 3,5000.00 + 4,375.00 - 6,000.00 = 33,375.00

S2 = S1 + S1 x i - P  = 33,375.00 + 33,375.00 * 0.125 - 6,000.00 = 33,375.00 + 4,171.88 - 6,000.00 = 31,546.88

S3 = S2 + S2 x i - P  = 31,546.88 + 31,546.88 * 0.125 - 6,000.00 = 31,546.88 + 3,943.36 - 6,000.00 = 29,490.23

S4 = S3 + S3 x i - P  = 29,490.23 + 29,490.23 * 0.125 - 6,000.00 = 29,490.23 + 3,686.28 - 6,000.00 = 27,176.51

S5 = S4 + S4 x i - P  = 27,176.51 + 27,176.51 * 0.125 - 6,000.00 = 27,176.51 + 3,397.06 - 6,000.00 = 24,573.58

S6 = S5 + S5 x i - P  = 24,573.58 + 24,573.58 * 0.125 - 6,000.00 = 24,573.58 + 3,071.70 - 6,000.00 = 21,645.28

S7 = S6 + S6 x i - P  = 21,645.28 + 21,645.28 * 0.125 - 6,000.00 = 21,645.28 + 2,705.65 - 6,000.00 = 18,350.93

S8 = S7 + S7 x i - P  = 18,350.93 + 18,350.93 * 0.125 - 6,000.00 = 18,350.93 + 2,293.87 - 6,000.00 = 14,644.80

Luis tendrá que pagar $14,644.80 al final de los tres años aparte de los 8 pagos que hizo de $6,000.00.
