Inversión anual

Caso 1
Tiempo que se necesita para ahorrar cierta cantidad de dinero.
Solución utilizando el modelo matemático

La Sra. Dávila necesita juntar $90,000.00 para cambiar su carro. Tiene ahorrado $30,000.00 y puede ahorrar $5,000.00 mensuales en una inversión que da una tasa de interés del 15% anual capitalizable cada mes.

¿Cuánto tiempo necesita para reunir la cantidad deseada?

En esta unidad se utiliza el modelo de interés compuesto, la única diferencia es que cada período se va a efectuar un pago que se tiene que agregar al saldo. El saldo para cada período va a ser igual al saldo del período anterior más los intereses del período más el pago que se efectua en dicho período. Se va a ir calculando el saldo para cada mes hasta que el saldo sea mayor o igual a $90,000.00 pesos.

Primer mes.
El saldo para el primer mes va a ser igual al saldo anterior $30,000.00 más los intereses del mes más el pago que se efectua cada período.
interés del mes = 30,000.00 * 0.0125
interés del mes = 375.00 pesos
El total para el siguiente mes es de $30,000.00 más $375.00 de interés más los $5,000.00 que va a ahorrar la Sra.
saldo para el primer mes = 30,000.00 + 375.00 + 5,000.00
saldo para el primer mes = 35,375.00
El saldo después de un mes va a ser de $35,375.00 pesos.
