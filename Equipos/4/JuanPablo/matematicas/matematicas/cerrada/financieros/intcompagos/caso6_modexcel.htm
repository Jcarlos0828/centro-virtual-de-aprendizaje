Inversión anual

Caso 1
Tiempo que se necesita para ahorrar cierta cantidad de dinero.
Solución utilizando la representación algebraica del modelo en Excel

La Sra. Dávila necesita juntar $90,000.00 para cambiar su carro. Tiene ahorrado $30,000.00 y puede ahorrar $5,000.00 mensuales en una inversión que da una tasa de interés del 15% anual capitalizable cada mes.

¿Cuánto tiempo necesita para reunir la cantidad deseada?

Para utilizar los elementos matemáticos que se definieron, se tiene que incluir el ahorro mensual que se va a estar haciendo, de manera que a la representación algebraica de interés compuesto St = St-1 +   St-1 * i se tiene que agregar lo que la Sra. Dávila va a ahorrar cada mes, que viene a ser el pago mensual (P) que está haciendo a su inversión, quedando la representación algebraicade la siguiente forma:
St = St-1 +   St-1 * i + P

Utilizando Excel se puede hacer una tabla utilizando la representación algebraica del modelo para facilmente encontrar el número de períodos que necesita la Sra. Dávila para ahorrar los $90,000.00:

TABLA EN EXCEL

Las acciones que se tomaron para generar esta tabla son las siguientes:
En la celda   B1   teclear   15%
En la celda   B2   teclear   =B1/12
En la celda   D4   teclear   30000
En la celda   B5   teclear   =D4 * $B$2         y llenar hacia abajo (Fill Down)
En la celda   C5   teclear   5000                      y llenar hacia abajo (Fill Down)
En la celda   D5   teclear   =D4 + B5 + C5    y llenar hacia abajo (Fill Down)

Para ver cuantos períodos se necesitan para tener los $90,000.00, se va haciendo Fill Down hasta que el saldo sea mayor o igual a dicha cantidad, en este caso se requieren 11 períodos para tener la cantidad que se desea. Una ventaja de utilizar Excel es que si la Sra. Dávila encuentra otro banco donde la tasa sea mejor, solo tiene que modificar la celda con la tasa de interés anual y buscar en la tabla para encontrar el número de períodos que se necesitan para obtener la cantidad deseada.

El número de meses que la Sra. Dávila necesita para ahorrar $90,000.00 son 11 y su saldo sería de $92,962.41 pesos.
