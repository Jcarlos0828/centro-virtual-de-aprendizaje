Valor futuro
Caso 1

Valor futuro de una anualidad a 4 años con una tasa de interés compuesto
anualizada capitalizable cada año.
Solución utilizando los elementos definidos

¿Cuál es el valor futuro de una anualidad de $15,000.00 por año pagaderos al final de cada año, durante 4 años a un interés compuesto de 8% capitalizable cada año?

S0      capital original.
i         tasa de interés del período en su forma decimal.
t         número de años o períodos.
P        pago que se realiza cada período.
St         saldo después de un t períodos de tiempo.

En el primer año el saldo inicial es de 0, así que no hay intereses durante el primer año, cuando finaliza el año se hace el primer pago de $15,000.00:
saldo inicial =  S0 = 0.00
intereses primer año = S0 x i = 0.00 x 0.08 = 0.00
pago primer año = P = 15,000.00
El saldo para el primer año va a ser igual al saldo anterior mas los intereses más el pago del primer año:
S1 = saldo primer año = S0 +  S0 x i + P
S1 = 0.00 + + 0.00 + 15,000.00
S1 = 15,000.00 pesos
El saldo del primer año va a ser de $15,000.00 pesos.

Para el segundo año el saldo va a ser igual al saldo anterior (15,000.00) más los intereses sobre ese saldo (15,000.00 x 0.08) mas el pago (15,000.00):
saldo anterior = S1 = 15,000.00
intereses segundo año = S1 x i = 15,000.00 x 0.08
intereses segundo año = 1,200.00
P = 15,000.00
S2 = saldo anterior + intereses del segundo año + pago segundo año
S2 = 15,000.00 + 1,200.00 + 15,000.00
S2 = 31,200.00 pesos
El saldo para el segundo año va a ser de $31,200.00 pesos.

Para el tercer año el saldo va a ser igual al saldo anterior (31,200.00) más los intereses sobre ese saldo (31,200.00 x 0.08) mas el pago (15,000.00):
saldo anterior = S2 = 31,200.00
intereses tercer año = S2 x i = 31,200.00 x 0.08
intereses tercer año = 2,496.00
P  = 15,000.00
S3 = saldo anterior + intereses tercer año + pago tercer año
S3 = 31,200.00 + 2,496.00 + 15,000.00
S3 = 48,696.00 pesos
El saldo para el tercer año va a ser de $48,696.00 pesos.

Para el cuarto año el saldo va a ser igual al saldo anterior ( 48,696.00) más los intereses sobre ese saldo ( 48,696 x 0.08) mas el pago (15,000.00):
saldo anterior =  S3 = 48,696.00
intereses cuarto año = S3 x i = 48,696 x 0.08
intereses cuarto año = 3,895.68
P = 15,000.00
S4 = saldo anterior + intereses cuarto año + pago cuarto año
S4 =  48,696 + 3,895.68 + 15,000.00
S4 = 67,591.68 pesos
El saldo para el cuarto año va a ser de $67,591.68 pesos.

El valor futuro de la anualidad es de $67,591.68 al finalizar el cuarto año.
