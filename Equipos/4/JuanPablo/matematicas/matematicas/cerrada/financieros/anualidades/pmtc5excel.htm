Valor futuro
Caso 5

Pago semestral de una anualidad para tener $50,000.00 en 5 años con una tasa de interés compuesto anualizada capitalizable cada semestre.
Solución utilizando la fórmula PMT de Excel

Si se quieren tener $50,000.00 en una cuenta de ahorros dentro de 5 años, encontrar cuánto se tiene que depositar semestralmente si la cuenta de ahorros paga el 8% anual con capitalización semestral.

Cuando solo se necesita saber el pago en forma directa se puede utilizar la siguiente función en Excel:

PMT (rate, nper, pv, fv, type)
Rate ->    tasa de interés para cada período
Nper ->   número de períodos
Pv ->       valor presente de la anualidad
Fv ->       valor futuro de la anualidad
Type ->    es 0 si el pago de la anualidad es al final del período y 1, si es al principio del período

Para este ejemplo los valores que se necesitan son:

Rate = 8% /2 = 0.04 - la capitalización es cada semestre
Nper = 5 * 2 = 10 - cada año tiene dos períodos
Pv = - se omite
Fv = 50000 - el valor futuro de la anualidad
Type = 0 - los pagos se van a hacer al final del período
=PMT (0.04, 10, ,50000, 0)
Lo que nos da una respuesta de $4,164.55 pesos.

Otra forma para calcular el pago es en Insert seleccionar Function , después Financial y seleccionar PMT.
Teclear en         Rate…….. 0.04
                         Nper......... 10
                         Pv.............                                no teclear nada
                         Fv............. 50000
                         Type......... 0

De manera que se necesitan depositar $4,164.55 cada semestre durante 5 años para lograr tener $50,000.00.
