Práctica de autoevaluación 2

  1. El Sr. Garza piensa ahorrar para su retiro dentro de 5 años. ¿Cuál es el valor futuro de una anualidad ordinaria de $25,000.00 por año, durante 5 años a un interés compuesto de 17% capitalizable cada año?

  1. Resolver el problema utilizando el modelo matemático.
  2. Resolver el problema utilizando el modelo con Excel.
  3. Si el Sr. Garza desea hacer los pagos al principio de cada año (anualidad anticipada), ¿cuál sería el valor futuro de la anualidad? Resolverlo utilizando el modelo con Excel. ¿cuánto sería la diferencia con la anualidad ordinaria?
  4. Hacer una gráfica para comparar la anualidad ordinaria y la anualidad anticipada (Pasos 2 y 3). ¿Qué es lo que se observa en la gráfica?
  5. Si el Sr. Garza decide pagar $30,000.00 en lugar de $25,000.00 cada año, ¿cuál es el valor futuro en los dos tipos de anualidades?
