Estimación de poblaciones de grupos aislados que forman un universo

Caso 1
Habitantes sajones, latinos y afro-americanos,
en un estado de los Estados Unidos.
Solución para un año, utilizando el modelo matemático.

Supóngase que en un estado de los Estados Unidos de América la población está integrada por sajones,latinos, y afroamericanos, y que cada grupo tiene un crecimiento diferente. Es claro que los porcentajes relativos de la población en cada uno de estos grupos va a cambiar a través del tiempo.
Hacer una estimación del número de elementos de cada uno de los grupos para el año 2020, si se tienen los siguientes datos:

En el año de 1995 la población está distribuida de la siguiente manera:
SAJONES                          2,750,000
LATINOS                           1,940,000
AFROAMERICANOS     1,680,000
TOTAL                              6,370,000

Las tasas de crecimiento de los grupos son:
SAJONES                           0.5%
LATINOS                          3.2%
AFROAMERICANOS     2.7%

Cada uno de los grupos sigue el comportamiento de un modelo de Crecimiento Poblacional Compuesto. La representación algebraica del modelo, de acuerdo a lo visto en el tema de Crecimiento Poblacional Compuesto, es:
Pt = Pt-1 + Pt-1*r o bien,
Pt = Pt-1( 1 + r )

Utilizando la relación anterior se puede encontrar, para cada uno de los grupos, una representación algebraica del modelo.
Para el grupo sajón, que tiene una tasa de crecimiento de 0.5% la representación algebraica sería:
St = St-1 + St-1* ( 0.005 ) o bien,
St = St-1 ( 1 + 0.005 )
St = St-1 (1.005 )

Para el grupo latino, que tiene una tasa de crecimiento de 3.2%, la representación algebraica es:
Lt = Lt-1 + Lt-1* ( 0.032 ) o bien,
Lt = Lt-1 ( 1 + 0.032 )
Lt = Lt-1 ( 1.032 )

Para el grupo de afroamericanos, que tiene una tasa de crecimiento de 2.7% la representación algebraica sería:
At = At-1 + At-1* ( 0.027 ) o bien,
At = At-1 ( 1 + 0.027 )
At = At-1 ( 1.027 )

Las tres relaciones anteriores son ecuaciones de diferencia finita que predicen el número de integrantes en cada grupo en años subsecuentes. Así, si se desea encontrar el número de integrantes del grupo sajón después de un año (en 1996) se tiene:
St = St-1 (1.005 ) al aplicar esta representación al primer año, queda la ecuación
S1 = S0 ( 1.005 ) sustituyendo S0 por su valor ( 2,750,000 ) se tiene,
S1 = 2,750,000 (1.005)
S1 = 2,763,750
La estimación del número de integrantes del grupo sajón para 1996 es de 2,763,750

El número de latinos después de un año (1996), se puede calcular de la manera siguiente:
Lt = Lt-1 ( 1.032 ) al aplicar esta representación al primer año queda la ecuación,
L1 = L0 ( 1.032) sustituyendo L0 por 1,940,000 se tiene,
L1 = 1,940,000 (1.032 )
L1 = 2,002,080
La estimación del número de integrantes del grupo latino para 1996 es de 2,002,080

Para calcular el número de habitantes afroamericanos después de un año ( 1996 ) se tiene que:
At = At-1 ( 1.027 ) al aplicar esta representación al primer año queda la ecuación,
A1 = A0 ( 1.027 ) sustituyendo A0 por su valor ( 1,680,000 ) se tiene,
A1 = 1,680,000 ( 1.027 )
A1 = 1,725,360
La estimación del número de habitantes afroamericanos para 1996 es de 1,725,360
