Deducción de la fórmula matemática para el modelo de
Crecimiento Poblacional Compuesto

Siguiendo el modelo del problema anterior con símbolos en lugar de números, se puede encontrar la fórmula para resolver problemas de Crecimiento Poblacional Compuesto.
Supongamos que se desea encontrar la población total Pt después de t años, si se tiene la población inicial P0 y una tasa anual r expresada en decimales.

El aumento de población durante el primer año (a1) se obtiene multiplicando la población inicial (P0), por la tasa de crecimiento anual expresada en decimales (r).
a1 = P0 x r

La población después de un año (P1), es la suma de la población inicial (P0), más el aumento que se tuvo durante el primer año (a1).
P1 = P0 + a1        sustituyendo a1 por su valor se tiene,
P1 = P0 + P0 x r   sacando P0 de factor común queda la ecuación,
P1 = P0 (1 + r)
*La población después de un año es --------------------------------- P1 = P0 (1 + r)

 El aumento de población durante el segundo año (a2) es el producto de multiplicar la población del año anterior (P1), por la tasa anual de crecimiento (r).
a2 = P1 x r    al sustituir P1 por su valor se tiene,
a2 = P0 (1 + r) x r

La población después de dos años (P2), será: la población del año anterior (P1), más el aumento producido durante el segundo año (a2).
P2 = P1 + a2    sustituyendo P1 y a2 por sus respectivos valores queda,
P2 = P0 (1 + r) + P0 (1 + r) x r       sacando de factor común P0 (1 + r) se tiene,
P2 = P0 (1 + r)(1 + r)     o bien,
P2 = P0 (1 + r)2
*La población después de dos años es------------------------------- P2 = P0 (1 + r)2

Para obtener el aumento de población durante el tercer año (a3) se multiplica la población del año anterior (P2), por la tasa anual de crecimiento (r).
a3 = P2 x r    sustituyendo P2 por su valor queda,
a3 = P0 (1 + r)2 x r

La población después de tres años (P3) viene a ser: la población del año anterior (P2), más el aumento producido durante el año (a3).
P3 = P2 + a3 al sustituir P2 y a3        por sus respectivos valores se tiene,
P3 = P0 (1 + r)2 + P0 (1 + r)2 x r     sacando de factor común P0 (1 + r)2 queda,
P3 = P0 (1 + r)2 (1 + r)       o bien,
P3 = P0 (1 + r)3
*La población después de tres años es-------------------------------- P3 = P0 (1 + r)3

Se puede observar que el exponente de (1 + r), es igual al número de años transcurridos después del año en que se da la población inicial, así el exponente 3, corresponde a la población después de tres años, el exponente 2 es para la población después de dos años.
Siguiendo este modelo, se puede deducir que la población total después de t años sería:
Pt = P0 (1 + r)t.
*La población total después de t años es------------------------------ Pt = P0 (1 + r)t

Se puede concluir entonces, que en un modelo de Crecimiento Poblacional Compuesto, la población total después de t años es:

Pt = P0 (1 + r)t

en donde,

Pt es la población total después de t años.
P0 es la población inicial.
r es la tasa de crecimiento anual, expresada en decimales.
t es el número de años después del año en que se toma la población inicial.
