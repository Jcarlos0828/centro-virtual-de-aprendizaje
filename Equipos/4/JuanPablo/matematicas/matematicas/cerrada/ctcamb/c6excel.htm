Encontrar la tasa de crecimiento para cada año y el porcentaje de cambio en las tasas.

Con Excel este problema se resuelve de forma muy rápida y sencilla.
Utilizando la fórmula  rt = (Pt / Pt-1) – 1  para  la tasa de crecimiento y el condicional IF con las fórmulas 
b = ( Rt / Rt-1 ) – 1  o b = 1 – ( Rt / Rt-1 )  para el porcentaje de cambio en las tasas, se genera la siguiente tabla:

TABLA  EN EXCEL

Para generar la tabla anterior se usaron la siguientes acciones:
En la celda   A2   teclear   1988
En la celda   A3   teclear   =A2 + 1 y llenar hacia abajo (Fill Down)
En la columna B   en las celdas de B2 a B11 vaciar la información de los datos del problema.
En la celda   C2   teclear   =((B3 / B2) – 1) * 100 y llenar hacia abajo (Fill Down)
En la celda   D3   usar el condicional IF. En Insert se busca Function y se selecciona la opción IF.
                                  En Logical test teclear       C2 < 0
                                  En Value if true teclear      (1 – (C3 / C2)) * 100
                                  y en Value if false teclear   ((C3 / C2) – 1) * 100

 

Como no se sabe si la tasa anterior es positiva o negativa hay que utilizar el condicional IF en la celda de D3. De esta forma si la tasa anterior es positiva usa la fórmula b = (Rt / Rt-1) – 1 y si es negativa utiliza la fórmula b = 1 – (Rt / Rt-1)
