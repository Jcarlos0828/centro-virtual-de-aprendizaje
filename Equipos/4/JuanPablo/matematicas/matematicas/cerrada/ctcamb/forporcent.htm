Deducción de fórmulas para encontrar
el porcentaje de cambio en la tasa de crecimiento

Para encontrar la tasa de crecimiento en cierto año t ( Rt ) cuando la tasa del año anterior ( Rt-1 ) es positiva se tenía la fórmula Rt = Rt-1 ( 1 + b ). En ésta fórmula se va a despejar b, que es el porcentaje de cambio en la tasa, expresado en decimales.
Rt = Rt-1 ( 1 + b )        al pasar Rt-1 dividiendo queda la ecuación,
Rt / Rt-1 = 1 + b           el 1 pasa restando y se tiene,
( Rt / Rt-1 ) – 1 = b      o bien,
b = ( Rt / Rt-1 ) – 1

Para encontrar la tasa de crecimiento en cierto año t ( Rt ) cuando la tasa del año anterior ( Rt-1 ) es negativa se tenía la fórmula Rt = Rt-1 ( 1 – b ). En ésta fórmula se va a despejar b.
Rt = Rt-1 ( 1 – b )         pasando Rt-1 dividiendo queda la ecuación,
Rt / Rt-1 = 1 – b el 1    pasa restando para tener
( Rt / Rt-1 ) – 1 = – b   cambiando los signos para tener b positiva se tiene
1 – ( Rt / Rt-1 ) = b      o bien,
b = 1 – ( Rt / Rt-1 )

Podemos concluir entonces que para encontrar el porcentaje de cambio en la tasa de crecimiento anual, se pueden usar las siguientes fórmulas, según el caso:

Si   Rt-1   es positiva se usa,
b = ( Rt / Rt-1 ) – 1

Si   Rt-1   es negativa se usa,
b = 1 – ( Rt / Rt-1 )

en donde,

Rt      es la tasa de crecimiento en el año t, expresada en porcentaje.
Rt-1   es la tasa de crecimiento en el año t – 1 , expresada en porcentaje.
b        es el porcentaje de cambio en la tasa, expresado en decimales.
