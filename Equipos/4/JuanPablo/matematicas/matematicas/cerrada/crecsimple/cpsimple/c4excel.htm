Estimación del número de Socios de un club
Solución utilizando Excel

El problema planteado anteriormente se puede resolver de una manera muy sencilla, ya que es posible determinar una ecuación fácilmente, que permita calcular el valor de la población en un futuro. Hay casos en donde el planteamiento del modelo no genera una ecuación sencilla de resolver, o bien situaciones en donde la complejidad del modelo es tal que ni siquiera permita obtener la ecuación final de un modelo planteado. En estos casos se tiene que recurrir a métodos computacionales para resolver el modelo que se está estudiando. Una herramienta que permite plantear modelos de manera sencilla es las denominadas "hoja de cálculo". Excel es un producto de este tipo.

El planteamiento en Excel del ejemplo anterior es el siguiente:

En una celda, por ejemplo A2, se registra el incremento annual de población. Se escogen dos columnas cualquiera para registrar en una el año y en la otra el valor de la población para ese año.

Por ejemplo, en la columna B se puede registrar el año y en la C la población. De esta manera el problema anterior se puede resolver haciendo B2 igual a 1998 y C2 igual a 45,000. la celda B3 se hace igual a la celda B2 mas uno. La celda C3 se hace igual a la celda C2 más la celda absoluta A2. Es importante registrar A2 como absoluta para que cuando se haga la siguiente operación las relaciones futuras hagan siempre referencia a ésta. Lo último que resta es propagar laas celdas B3 y C3 hacia abajo hasta llegar al año 2015. Haciendo esto la tabla generada en Excel es la siguiente:
