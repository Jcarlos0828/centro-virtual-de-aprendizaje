Caso 1
Estimación del número de libros que se venderán en cierto año
Solución utilizando la fórmula

Un libro de texto es editado anualmente. En el año 1998 se vendieron 45,000 ejemplares y sabemos que las ventas se incrementan en 450 ejemplares por año. ¿Cuántos libros se espera que se vendan en el año 2015?

En este caso se extendió el número de años del 2003 al 2015. Utilizando la fórmula de Crecimiento Poblacional Simple Pt = P0 +  t*a  para resolver este problema, se tiene:

Para calcular lo anterior, primero hay que encontrar el número de años que se está prediciendo hacia el futuro, que en este caso es 2015 – 1998 que viene a ser 17 años.
Datos:
P0 = 45,000
a = 450
t = 17 años
P17 = ? se busca el número de libros después de 17 años.
Solución:
Pt = P0 +  t*a      aplicando la fórmula para 17 años queda la ecuación,
P17 = P0 + (17* 450)
P17 = 45,000 + 7,650
P17 = 52,650
