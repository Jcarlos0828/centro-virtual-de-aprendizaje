Ventas con variación en el precio

Caso 3
Ventas de un paquete computacional
Solución utilizando fórmula.

La compañía de software Softmex ha firmado un contrato con una dependencia de gobierno para la venta de un paquete computacional. En el contrato, Softmex se compromete a proporcionar 1500 unidades del paquete en el año 2000 y a incrementar el número de unidades a razón del 10% anual. El precio del paquete es de $250 dólares. El dólar se cotiza en 10.10 pesos al inicio del 2000, y se estima que su valor se incremente en un 15% anual. Calcule las ventas del paquete para el año 2008 en pesos.

Datos:
U2000 = 1500 unidades
r = 10%      incremento anual de unidades
P2000 = 250 x 10.10 = $2,525 pesos
p = 15%      incremento anual del precio
n = 2000
t = 2008
V2008 = ¿?   se desean conocer las ventas en pesos para el 2008

Solución:
De acuerdo a la fórmula Vt = Un(1 + r)(t-n) Pn (1 + p)(t-n)      aplicada a los datos:
V2008 = U2000(1 + r)(2008 - 2000) P2000(1 + p)(2008 - 2000)
V2008 = (1500)(1 + 0.1)(8) (2525)(1+0.15)(8)
V2008 = (1500)(1.1)8 (2525)(1.15)8
V2008 = (1500)( 2.1436) (2525)( 3.0590229)
V2008 = (3215) (7724.032728)
V2008 = 24,832,765.22
Las ventas para el 2008 son de $24,832,765.22 pesos.

* Se puede observar que el número de unidades Un(1 + r)(t-n) se redondeó antes de multiplicarlo por el precio, para no manejar unidades fraccionarias; quedando de 3215.383215 a 3215
