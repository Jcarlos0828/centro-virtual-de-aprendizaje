Estimación de ventas con crecimiento compuesto.

Caso 2
Ventas de una empresa de servicios de internet
Solución para 4 meses utilizando la representación algebráica

Se conoce que el número de usuarios de Conectix, una empresa proveedora de servicios de internet, se incrementa en un 35% mensual. La renta mensual que paga cada usuario por el servicio es de $250 pesos. Si en junio de 1999 se tenían 1346 usuarios, ¿cuáles serán los ingresos por ventas que se tendrán en diciembre de 1999?

El modelo matemático para este caso establece que las ventas en cualquier mes se pueden obtener multiplicando el número de usuarios por el pago mensual por usuario (250 pesos).
El número de usuarios en cualquier año t (Ut) es igual al número de usuarios del mes anterior (Ut-1) mas el incremento de usuarios en el mes. El incremento en el número de usuarios es el producto de multiplicar el número de usuarios del mes anterior(Ut-1) por la tasa de crecimiento expresada en decimales(r = 0.35).

Como se puede observar el incremento en el número de usuarios, guarda el comportamiento de un Crecimiento Compuesto. La representación algebraica de un modelo de Crecimiento Poblacional Compuesto es Pt = Pt-1+ Pt-1* r o bien, sacando Pt-1 de factor común Pt = Pt-1(1 + r).

La representación algebraica del modelo de Crecimiento Compuesto para encontrar el número de usuarios en cualquier período t puede ser escrita de la siguiente manera.
Ut = Ut-1+ Ut-1* r o bien,
Ut = Ut-1(1 + r).

Mes de junio.
Para encontrar las ventas del mes de junio se multiplica el número de usuarios del mes (1346) por la renta mensual que paga cada usuario (250 pesos).
Ventas de junio = 1,346 x 250
Ventas de junio = 336,500 pesos
Las ventas del mes de junio son de 336,500 pesos

Mes de julio.
Para calcular las ventas del mes de julio primero se tiene que encontrar el número de usuarios del mes. Para hacerlo se puede utilizar la representación algebraica del modelo de Crecimiento Compuesto
Ut = Ut-1+ Ut-1* r o bien, Ut = Ut-1(1 + r), que aplicada al mes de julio queda:
Ujulio = Ujunio(1 + r)             Ujunio = 1346 y r = 0.35  sustituyendo estos valores se tiene que,
Ujulio = 1346( 1 + 0.35)
Ujulio = 1346(1.35)
Ujulio =1817
En el mes de julio hay 1,817 usuarios.
Las ventas del mes son el producto de multiplicar el número de usuarios del mes (1,817)por el pago mensual por usuario (250 pesos).
Ventas de julio = 1,817 x 250
Ventas de julio = 454,250
Las ventas del mes de julio son de 454,250 pesos.

Mes de agosto.
Para encontrar el número de usuarios del mes se utiliza la representación algebraica de un Crecimiento Compuesto Ut = Ut-1(1 + r), aplicada al mes de agosto.
Uagosto = Ujulio(1 + r)              Ujulio= 1,817 y r = 0.35 sustituyendo estos valores queda la ecuación
Uagosto = 1,817 ( 1 + 0.35 )
Uagosto =1,817 (1.35)
Uagosto = 2,453
El número de usuarios en el mes de agosto es de 2,453.
Para encontrar las ventas del mes se multiplica el número de usuarios del mes (2,4539por la renta mensual por usuario (250).
Ventas de agosto = 2,453 x 250
Ventas de agosto = 613, 250
Las ventas del mes de agosto son de 613,250 pesos.

Mes de septiembre.
El número de usuarios del mes de septiembre se obtiene utilizando la representación algebraica de Crecimiento Compuesto Ut = Ut-1(1 + r), aplicada al mes de septiembre.
Usep. = Uagosto(1 + r)                  Uagosto = 2,453 y r 0 0.35 sustituyendo estos valores se tiene,
Usep. = 2,453 ( 1 + 0.35 )
Usep. = 2,453 ( 1.35 )
Usep. = 3,312
El número de usuarios del servicio en el mes de septiembre es de 3,312.
Las ventas del mes son el producto de multiplicar el número de usuarios del mes(3,312) por el precio del servicio por usuario (250 pesos).
Ventas de septiembre = 3,312 x 250
Ventas de septiembre = 828,000
Las ventas del mes de septiembre son de 828,000 pesos.
