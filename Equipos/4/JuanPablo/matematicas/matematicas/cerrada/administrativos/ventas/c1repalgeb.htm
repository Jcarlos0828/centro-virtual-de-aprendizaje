Representación algebraica del modelo simple de ventas

El modelo matemático para este caso establece que para calcular las ventas en cualquier año t (Vt), se multiplica el número de unidades vendidas en el año ( Ut) por el precio por unidad (P).
Vt = Ut x P            para t = 1,2,3,…….
El número de unidades vendidas en el año t ( Ut) es igual al número de unidades que se vendieron el año anterior ( Ut-1) más el aumento anual en este número ( i ).
Ut = Ut-1 + i           para t = 2,3,4,……….

Las ecuaciones Vt = Ut x P y Ut = Ut-1 + i , determinan la representación algebraica del modelo matemático.
