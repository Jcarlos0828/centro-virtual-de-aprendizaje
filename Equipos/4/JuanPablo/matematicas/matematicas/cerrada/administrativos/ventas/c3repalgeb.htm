Representación algebraica del modelo

El modelo en este caso sería:
Las ventas en cualquier año t mayor que 1 (Vt) son el producto de multiplicar el número de unidades vendidas en el año (Ut) por el precio en pesos, por cada unidad (Pt).
Vt = Ut * Pt

El aumento en el número de unidades vendidas guarda el comportamiento de un Crecimiento Compuesto y por lo tanto se puede utilizar la representación algebraica del modelo Pt = Pt-1 ( 1 + r ). Aplicando esta representación para encontrar el número de unidades vendidas en el año t (Ut) se llega a la ecuación
Ut = Ut-1 ( 1 + r )      ya que el número de unidades aumenta en un 10% anual, el valor de r es de 0.10. Sustituyendo este valor en la ecuación se llega a
Ut = Ut-1 ( 1 + 0.10 )
Ut = Ut-1 ( 1.10 )

El precio por unidad es de 250 dólares, pero la cotización del dólar en pesos tiene un incremento de un 15% anual. Esto representa también un Crecimiento Compuesto, por lo tanto, si tomamos Pt como el precio por unidad en pesos, este sería igual al precio del año anterior ( Pt-1 ) mas el aumento de precio en el año
( Pt-1 * 0.15 )
Pt = Pt-1 + Pt-1 * 0.15
Pt = Pt-1 ( 1 + 0.15 )
Pt = Pt-1 ( 1.15 )

Este sistema de tres ecuaciones
Vt = Ut * Pt ,
Ut = Ut-1 ( 1+ 0.10 ) y
Pt = Pt-1 ( 1 + 0.15 ),
constituye la representación algebraica del modelo para este caso específico.
