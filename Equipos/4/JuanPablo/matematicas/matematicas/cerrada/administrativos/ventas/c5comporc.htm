Calcule las ventas totales y el porcentaje de ventas de cada tipo de teléfono para el primer trimestre del 2000, suponiendo que los precios se mantienen constantes.

En este tipo de casos es muy conveniente calcular el porcentaje de distribución de las ventas totales por el tipo de producto, en este caso los teléfonos, en el período inicial y en el final, para posteriormente hacer una comparación entre ellos.

Primer trimestre de 1999 (período 0):
En el primer trimestre de 1999 las ventas estaban distribuidas de la siguiente manera:
Teléfono tipo A        $550,000
Teléfono tipo B     $1,313,200
Teléfono tipo C        $733,500

Ventas Totales      $2,596,700

Para obtener los porcentajes de distribución de las ventas por tipo de teléfono, se dividen las ventas de cada tipo entre el total de ventas. El resultado se multiplica por 100 para que quede expresado en porciento y no en decimales.

Teléfono tipo A               550000 / 2596700  = 0.2118..................  21.18%
Teléfono tipo B             1313200 / 2596700 = 0.5057..................  50.57%
Teléfono tipo C               733500 / 2596700  = 0.2825..................  28.25 %

Segundo trimestre de 1999 (período 1):
En el segundo trimestre de 1999 las ventas estaban distribuidas de la siguiente manera:
Teléfono tipo A          $302,000
Teléfono tipo B       $1,812,350
Teléfono tipo C          $918,000

Ventas Totales        $3,032,350

Dividiendo las ventas de cada tipo de teléfono entre el total de ventas se obtienen los porcentajes de distribución de las ventas para el segundo trimestre. Éstos son:

Teléfono tipo A             302000 / 3032350  = 0.0996...................    9.96%
Teléfono tipo B             1812350 / 3032350 = 0.5977...................  59.77%
Teléfono tipo C                918000 / 3032350 = 0.3027...................  30.27%

Tercer trimestre de 1999 (período 2):
En el tercer trimestre de 1999 las ventas estaban distribuidas de la siguiente manera:
Teléfono tipo A            $166,000
Teléfono tipo B        $2,502,450
Teléfono tipo C        $1,147,500

Ventas Totales         $3,815,950

Dividiendo las ventas de cada tipo de teléfono entre el total de ventas se obtienen los porcentajes de distribución de las ventas para el segundo trimestre. Éstos son:

Teléfono tipo A               166000 / 3815950 = 0.0435.....................   4.35%
Teléfono tipo B            2502450 / 3815950 = 0.6568..................... 65.58%
Teléfono tipo C            1147500 / 3815950 = 0.3007..................... 30.07%
