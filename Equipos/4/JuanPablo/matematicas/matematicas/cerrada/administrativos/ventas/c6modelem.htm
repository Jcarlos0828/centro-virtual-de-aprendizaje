Determinación de precio en base a un nivel de ventas

Caso 6
Precios de los productos de una tienda de helados
Solución utilizando el modelo matemático con los elementos de ventas.

Una tienda de helados vende solamente helados de crema en presentación doble. Ya a que las ventas de helados tienen temporalidad debido a que éstos son más populares en los meses de verano, el gerente tiene pensado empezar a vender pastel en rebanadas para tratar de contrarrestar las ventas bajas en invierno. Se conocen los datos de 1998 y se quieren fijar precios para todo el año de 1999. Se vendieron un total de 17424 helados en 1998 a un precio de $13. Se conoce que el número de helados vendidos se incrementa en un 14% anual y se espera vender la mitad de pasteles que de helados. Al incluir los pasteles en 1999 como otro producto se desea que las ventas totales se incrementen en un 97% con respecto a 1998.

El precio de un helado y un pastel juntos debe ser de $30. ¿Qué precios deben tener los helados y los pasteles para lograr los objetivos de ventas?

Año 1998.
En el año de 1998 solamente se vendieron helados. Para calcular las ventas del año (V1998) se multiplica el número de helados vendidos en el año (H1998 =17424) por el precio de cada helado (P =13 pesos).
V1998 = H1998 x P sustituyendo los valores de U1998 y de P se llega a
V1998 = 17,424 x 13
V1998 = 226,512
Las ventas del año 1998 son de $ 226,512 pesos.

Año 1999.
Del año 1999 se necesitan los siguientes datos:
El número de helados que se venden en el año.
El número de pasteles que se venden en el año.
Las ventas totales esperadas para el año.

Número de helados que se venden en el año.
Se sabe que el número de helados vendidos se incrementa en un 14% anual. Esto corresponde a un Crecimiento Compuesto y se puede utilizar la representación algebraica Pt = Pt-1(1 + r). Aplicándola al número de helados se puede escribir como Ht = Ht-1(1 + r), para 1999 se tendría la ecuación.
H1999 = H1998(1 + r) H1998=17,424 y r = 0.14 al sustituir estos valores se tiene
H1999 = 17,424 (1 +0.14)
H1999 = 17,424(1.14)
H1999 = 19,863.
El número de helados que se espera vender en 1999 es de 19,863.

Número de pasteles que se venden en el año.
De acuerdo a los datos del caso, el número de pasteles que se espera vender en el año, es la mitad del número de helados que se venden en el año (19,863).
El numero de pasteles es igual al número de helados entre 2.
P1999 = H1999 / 2
P1999 = 19,863 / 2
P1999 = 9931.5
Se espera vender 9,932 pasteles en el año.

Ventas totales del año.
Se espera que las ventas totales del año se incrementen en un 97%. Volvemos a tener un Crecimiento Compuesto, por lo tanto se utiliza la representación algebraica Pt = Pt-1(1 + r) aplicada a ventas en el año 1999 y con una tasa de 97%.
V1999 = V1998(1 + 0.97) V1998 = 226,512 de acuerdo a lo calculado anteriormente
V1999 = 226,512(1.97)
V1999 = 446,229
Las ventas esperadas para 1999 son de $ 446,229 pesos.

Ecuaciones.
Para calcular las ventas totales del año (V1999) se suman las ventas por helados más las ventas por pasteles.
V1999 = VH’99 + VP’99
Las ventas por helados se calculan como la multiplicación del número de helados por el precio.
VH’99 = UH’99 x PH’99
Las ventas por pasteles se calculan como la multiplicación del número de pasteles por el precio.
VP’99 = UP’99 x PP’99
Sustituyendo las ventas por helados y por pasteles en la fórmula de ventas totales queda:
V1999 = (UH’99 x PH’99) + (UP’99 x PP’99)
Sustituyendo los datos que se encontraron:
V1999 = (UH’99 x PH’99) + (UP’99 x PP’99)
446229 = (19863 x PH’99) + (9932 x PP’99)

En la descripción del caso también se menciona que el precio combinado de un helado y un pastel debe ser igual a 30 pesos, por lo que se puede escribir:
PH’99 + PP’99 = 30

El sistema de ecuaciones a resolver esta formado por:
(19863 x PH’99) + (9932 x PP’99) = 446229
PH’99 + PP’99 = 30

Para poder encontrar la solución del caso se puede despejar la ecuación para PP’99:
PP’99 = 30 – PH’99

Sustituyendo en la ecuación de ventas:
19863 PH’99    +             9932 PP’99                 = 446229
19863 PH’99    +          9932 (30 – PH’99)         = 446229
19863 PH’99    +     297960   – 9932 PH’99      = 446229
Pasando los términos constantes de un lado de la ecuación y los que tienen la variable PH’99 del otro lado:
19863 PH’99    –   9932 PH’99     =    446229   –  297960
9931 PH’99   =   148269
PH’99    =   148269 / 9931
PH’99    =   14.93 pesos
El precio de los helados para 1999 debe ser de $14.93 pesos

Sustituyendo en la ecuación despejada para PP’99:
PP’99 = 30 – PH’99
PP’99 = 30 – 14.93
PP’99 = $15.07 pesos
El precio de los pasteles para 1999 debe ser de $15.07 pesos
