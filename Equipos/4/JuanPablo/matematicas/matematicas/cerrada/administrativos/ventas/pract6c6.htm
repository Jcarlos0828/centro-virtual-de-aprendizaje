La empresa "Goldsun" construye monitores de 14 pulgadas para uso con computadoras.  El precio de cada monitor es de $1699 pesos.  Se conoce que se vendieron 276 monitores durante 1998, y este número se incrementa en un 20% anual.   Para 1999 se tiene planeado el lanzamiento de monitores de 17 pulgadas.  Se sabe que se venderá aproximadamente el 60% de unidades de 17 pulgadas con respecto al número de unidades de 14 pulgadas.  Se desea establecer el precio mínimo de estas unidades y el nuevo precio de los monitores de 14 pulgadas en base a un nivel de ventas deseado.  Si se desea tener el doble de las ventas de 1998 en 1999 y el precio del monitor de 17 pulgadas debe ser igual 75% mayor al de 14 pulgadas, encuentre los precios mínimos a manejar en 1999 para obtener el nivel de ventas deseado.
  1. Construya el modelo matemático para plantear las ecuaciones de la solución.
  2. Utilice un archivo en Excel en el que utilice la herramienta Goal Seek para obtener la solución.
