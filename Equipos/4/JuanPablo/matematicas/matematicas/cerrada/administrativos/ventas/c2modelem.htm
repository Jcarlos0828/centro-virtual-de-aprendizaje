Estimación de ventas con crecimiento compuesto.

Caso 2
Ventas de una empresa de servicios de internet
Solución utilizando el modelo con los elementos

La empresa Conectix, que es proveedora de servicios de internet, conoce que el número de usuarios que contratan sus servicios se incrementa en un 35% mensual. La renta mensual que paga cada usuario por el servicio es de $250 pesos. Si en junio de 1999 se tenían 1346 usuarios, ¿cuáles serán los ingresos por ventas que se tendrán en diciembre de 1999?

De acuerdo al modelo matemático, las ventas en cierto período t (Vt), son el producto de multiplicar el número de usuarios del período (Ut) por el precio de la renta por usuario(P). El incremento en el número de usuarios guarda el comportamiento de un Crecimiento Compuesto, por lo tanto, para encontrar el número de usuarios en cierto período t (Ut), se utiliza la representación algebraica del modelo de Crecimiento Compuesto Ut = Ut-1+ Ut-1* r o bien, Ut = Ut-1(1 + r).

Mes de junio.
Las ventas en el mes de junio (V6) son el producto de multiplicar el número de usuarios del mes (U6 = 1,346), multiplicado por el precio de la renta por usuario (P = 250 pesos).
V6 = U6 x P                 sustituyendo los valores de U6 y de P se llega a la ecuación,
V6 = 1,346 x 250
V6 = 336,500
Las ventas del mes de junio ascienden a 336,500 pesos.

Mes de julio.
Para obtener las ventas del mes de julio primero hay que encontrar el número de usuarios del mes (Ut). Para hacerlo se utiliza la ecuación:
Ut = Ut-1(1 + r)         aplicando la ecuación al mes 7 se tiene
U7 = U6(1 + r)          al sustituir los valores de U6 = 1,346 y de r = 0.35 se llega a
U7 = 1,346(1+0.35)
U7 = 1,346(1.35)
U7 = 1,817
En el mes de julio hay 1,817 usuarios del servicio.
Para obtener las ventas del mes de julio (V7) se multiplica el número de usuarios del mes (U7 = 1,817) por la renta mensual por usuario(P = 250 pesos).
V7 = U7 x P                    sustituyendo los valores de U7 y de P se llega a la ecuación,
V7 = 1,817 x 250
V7 = 454,250
Las ventas del mes de julio ascienden a 454,250 pesos.

Mes de agosto.
El número de usuarios para agosto se obtiene con la ecuación Ut = Ut-1(1 + r) aplicada para el octavo período.
U8 = U7(1 + r)             U7 = 1,817 y r = 0.35 al sustituir estos valores queda la ecuación,
U8 = 1,817 ( 1+ 0.35 )
U8 = 1,817(1.35)
U8 = 2,453
El número de usuarios en el mes de agosto es de 2,453.
Las ventas del mes son el producto de multiplicar el número de usuarios del mes (U8 = 2,453) por la renta mensual del servicio por usuario (P = 250 pesos).
V8 = U8 x P             sustituyendo U8 y P por sus respectivos valores se llega a la ecuación,
V8 = 2,453 x 250
V8 =613,250
Las ventas del mes de agosto ascienden a 613,250 pesos.

Mes de septiembre.
El número de usuarios del mes de septiembre se calcula con la ecuación:
U9 = U8(1 + r)         sustituyendo los valores de U8 = 2,453 y r =0.35 queda,
U9 = 2,453 ( 1 + 0.35)
U9 = 2,453(1.35)
U9 = 3,312
El número de usuarios en el mes de septiembre es de 3,312.
Las ventas del mes se calculan multiplicando los usuarios del mes (U9 = 3,312) por la renta mensual por usuario (P = 250 pesos).
V9 = U9 x P
V9 = 3,312 x 250
V9 = 828,000
Las ventas correspondientes al mes de septiembre ascienden a 828,000 pesos.
