Estimación de ventas con crecimiento simple.

Caso 1
Ventas de una empresa fabricante de equipo de cómputo
Solución utilizando fórmula.

La empresa Compuquick dedicada a la fabricación y venta de equipo de cómputo firmó un contrato de venta a cinco años con una institución educativa. En el contrato, Compuquick se compromete a vender 500 computadoras el primer año y a incrementar este número en 100 unidades anualmente. El precio de venta pactado es de $1000 dólares por computadora. Calcule las ventas anuales para los siguientes 5 años.

Solución para encontrar las ventas del 5to. año
Datos:
U1 = 500 unidades
i = 100 unidades
t = 5 años
n = 1
P = $1000 dólares por unidad
V5 = ¿? se desean conocer las ventas en dólares para el año 5

Solución:
De acuerdo a la fórmula Vt = [Un + (t-n)i] P aplicada para 5 años con período inicial 1, se tiene la ecuación:
V5 = [U1 + (5-1)i] P
V5 = [500 + (5-1)100] 1000
V5 = [500 + (4)100] 1000
V5 = [500 + 400] 1000
V5 = [900] 1000
V5 = 900,000
Las ventas del 5to. año ascienden a  $900,000 dólares.
