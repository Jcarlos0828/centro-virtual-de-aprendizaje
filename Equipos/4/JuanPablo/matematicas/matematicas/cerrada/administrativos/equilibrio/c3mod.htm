Caso 3
Punto de equilibrio para una fábrica de cosméticos
Solución para 3 meses utilizando el modelo matemático con los elementos.

Una maquiladora que fabrica lápices labiales para una prestigiada marca de cosméticos en Europa tiene costos fijos mensuales de $200,000 pesos.  El costo variable unitario es de $75 pesos y el precio de cada lápiz labial es de $150 pesos.  Encuentre el punto de equilibrio en unidades y también en pesos para esta situación.  Evalúe el cambio que tendrían en el punto de equilibrio un aumento y una disminución de 20% en el precio.

El punto de equilibrio se encuentra al calcular el número de unidades producidas y vendidas que hacen que no se tengan ganancias o pérdidas, o sea que las utilidades tengan un valor de cero.  En este caso, los ingresos (o ventas)  se cancelan con los egresos (o costos).  La fórmula de utilidades para un período t, establece que las utilidades (UTt) son iguales a las ventas (Vt) menos los costos totales (CTt):
UTt = Vt -CTt

Para el caso actual:
Las ventas (Vt) se pueden calcular como el número de unidades vendidas (Ut) multiplicado por el precio (150):
Vt = 150 Ut

Los costos totales (CTt) se pueden expresar como los costos fijos (200000) más el costo variable unitario (75) multiplicado por el número de unidades vendidas (Ut):
CTt = 200000 + 75 Ut

Las utilidades (UTt) son iguales a las ventas (150 Ut) menos los costos totales (200000 + 75 Ut):
UTt = 150 Ut - (200000 + 75 Ut)

Simplificando y agrupando términos comunes:
UTt = 150 Ut - 200000 - 75 Ut
UTt = 75 Ut - 200000

El punto de equilibrio se encuentra al igualar las utilidades a cero y despejar para encontrar el número de unidades:
UTt = 75 Ut - 200000 = 0
75 Ut - 200000 = 0
75 Ut = 200000
Ut = 200000/75 = 2667 unidades.
En pesos:
Vt = Ut x P = 2667 x 150 = 400050
Para la situación actual, el punto de equilibrio se encuentra en 2667 unidades o $400,050 pesos.

Si se el precio aumenta en 20%:
El precio entonces sería igual al precio original (150) multiplicado por uno más el porcentaje de incremento (1.2):
P = 150 x 1.2
P = 180
Las ventas (Vt) se pueden calcular como el número de unidades vendidas (Ut) multiplicado por el precio (180):
Vt = 180 Ut

Los costos totales (CTt) se calculan igual que en la situación actual:
CTt = 200000 + 75 Ut

Las utilidades (UTt) son iguales a las ventas (180 Ut) menos los costos totales (200000 + 75 Ut):
UTt = 180 Ut - (200000 + 75 Ut)

Simplificando y agrupando términos comunes:
UTt = 180 Ut - 200000 - 75 Ut
UTt = 105 Ut - 200000

El punto de equilibrio se encuentra al igualar las utilidades a cero y despejar para encontrar el número de unidades:
UTt = 105 Ut - 200000 = 0
105 Ut - 200000 = 0
105 Ut = 200000
Ut = 200000/105 =  1905 unidades.
En pesos:
Vt = Ut x P = 1905 x 150 = 285750
Si el precio aumenta en 20%, el punto de equilibrio se encuentra en 1905 unidades o $285,750 pesos.

Si el precio disminuye en 20%:
El precio entonces sería igual al precio original (150) multiplicado por uno menos el porcentaje de incremento, o (1.2):
P = 150 x 1.2
P = 120
Las ventas (Vt) se pueden calcular como el número de unidades vendidas (Ut) multiplicado por el precio (120):
Vt = 120 Ut

Los costos totales (CTt) se calculan igual que en la situación actual:
CTt = 200000 + 75 Ut

Las utilidades (UTt) son iguales a las ventas (120 Ut) menos los costos totales (200000 + 75 Ut):
UTt = 120 Ut - (200000 + 75 Ut)

Simplificando y agrupando términos comunes:
UTt = 120 Ut - 200000 - 75 Ut
UTt = 45 Ut - 200000

El punto de equilibrio se encuentra al igualar las utilidades a cero y despejar para encontrar el número de unidades:
UTt = 45 Ut - 200000 = 0
45 Ut - 200000 = 0
45 Ut = 200000
Ut = 200000/45 =  4444 unidades.
En pesos:
Vt = Ut x P = 4444 x 150 = 666600
Si el precio disminuye en 20%, el punto de equilibrio se encuentra en 4444 unidades o $666,600 pesos.
