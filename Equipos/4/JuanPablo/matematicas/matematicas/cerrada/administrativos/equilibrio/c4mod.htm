Punto de equilibrio para una compañía farmacéutica
Solución para 3 trimestres utilizando el modelo matemático con los elementos.

Laboratorios Berck, una compañía farmacéutica, tiene la fórmula exclusiva de un nuevo medicamento antiviral que acaba de introducir al mercado.  A continuación se muestran los costos fijos, costo variable unitario y el precio, así como sus tasas de incremento trimestrales correspondientes:

tasa de incremento trim.

Costo var. unit.

$56

Precio

$295

Encuentre los puntos de equilibrio en unidades y pesos para los primeros cuatro trimestres del año.

El punto de equilibrio se encuentra al calcular el número de unidades producidas y vendidas que hacen que no se tengan ganancias o pérdidas, o sea que las utilidades tengan un valor de cero.  En este caso, los ingresos (o ventas)  se cancelan con los egresos (o costos).  La fórmula de utilidades para un período t, establece que las utilidades (UTt) son iguales a las ventas (Vt) menos los costos totales (CTt):
UTt = Vt -CTt

Para el trimestre 1:
Las ventas (Vt) se pueden calcular como el número de unidades vendidas (Ut) multiplicado por el precio (295):
Vt = 295 Ut

Los costos totales (CTt) se pueden expresar como los costos fijos (300000) más el costo variable unitario (56) multiplicado por el número de unidades vendidas (Ut):
CTt = 300000 + 56 Ut

Las utilidades (UTt) son iguales a las ventas (295 Ut) menos los costos totales (300000 + 56 Ut):
UTt = 295 Ut - (300000 + 56 Ut)

Simplificando y agrupando términos comunes:
UTt = 295 Ut - 300000 - 56 Ut
UTt = 239 Ut - 300000

El punto de equilibrio se encuentra al igualar las utilidades a cero y despejar para encontrar el número de unidades:
UTt = 239 Ut - 300000 = 0
239 Ut - 300000 = 0
239 Ut = 300000
Ut = 300000/239 =  1255 unidades.
En pesos:
Vt = Ut x P = 1255 x 295 = 370225
Para el trimestre 1, el punto de equilibrio se encuentra en 1255 unidades o $370,225 pesos.

Para el trimestre 2:
El precio se incrementa en 6%, entonces sería igual al precio del trimestre anterior (295) multiplicado por uno más el porcentaje de incremento (1.06):
P = 295 x 1.06
P = 312.7
Las ventas (Vt) se pueden calcular como el número de unidades vendidas (Ut) multiplicado por el precio (312.7):
Vt = 312.7 Ut

Los costos fijos se incrementan en 5.1% por lo que el valor de éstos es igual al de los costos fijos del trimestre anterior (300000) multiplicado por uno más el porcentaje de incremento (1.051):
CF = 300000 x 1.051 = 315300
El costo variable unitario se incrementa en 4.3% por lo que su valor es igual al del costo variable unitario del trimestre anterior (56) multiplicado por uno más el porcentaje de incremento (1.043):
CV = 56 x 1.043 = 58.41
Los costos totales (CTt) se pueden expresar como los costos fijos (315300) más el costo variable unitario (58.41) multiplicado por el número de unidades vendidas (Ut):
CTt = 315300 + 58.41 Ut

Las utilidades (UTt) son iguales a las ventas (312.7 Ut) menos los costos totales (315300 + 58.41 Ut):
UTt = 312.7 Ut - (315300 + 58.41 Ut)

Simplificando y agrupando términos comunes:
UTt = 312.7 Ut - 315300 - 58.41 Ut
UTt = 254.29 Ut - 315300

El punto de equilibrio se encuentra al igualar las utilidades a cero y despejar para encontrar el número de unidades:
UTt = 254.29 Ut - 315300 = 0
254.29 Ut - 315300 = 0
254.29 Ut = 315300
Ut = 315300/254.29 = 1240  unidades.
En pesos:
Vt = Ut x P = 1240 x 312.7 = 387748
Para el trimestre 2, el punto de equilibrio se encuentra en 1240 unidades o $387,748 pesos.

Para el trimestre 3:
El precio se incrementa en 6%, entonces sería igual al precio del trimestre anterior (312.7) multiplicado por uno más el porcentaje de incremento (1.06):
P = 312.7 x 1.06
P = 331.46
Las ventas (Vt) se pueden calcular como el número de unidades vendidas (Ut) multiplicado por el precio (331.46):
Vt = 331.46 Ut

Los costos fijos se incrementan en 5.1% por lo que el valor de éstos es igual al de los costos fijos del trimestre anterior (315300) multiplicado por uno más el porcentaje de incremento (1.051):
CF = 315300 x 1.051 = 331380.3
El costo variable unitario se incrementa en 4.3% por lo que su valor es igual al del costo variable unitario del trimestre anterior (58.41) multiplicado por uno más el porcentaje de incremento (1.043):
CV = 58.41 x 1.043 = 60.92
Los costos totales (CTt) se pueden expresar como los costos fijos (331380.3) más el costo variable unitario (60.92) multiplicado por el número de unidades vendidas (Ut):
CTt = 331380.3 + 60.92 Ut

Las utilidades (UTt) son iguales a las ventas (331.46 Ut) menos los costos totales (331380.3 + 60.92 Ut):
UTt = 331.46 Ut - (331380.3 + 60.92 Ut)

Simplificando y agrupando términos comunes:
UTt = 331.46 Ut - 331380.3 - 60.92 Ut
UTt = 270.54 Ut - 331380.3

El punto de equilibrio se encuentra al igualar las utilidades a cero y despejar para encontrar el número de unidades:
UTt = 270.54 Ut - 331380.3 = 0
270.54 Ut - 331380.3 = 0
270.54 Ut = 331380.3
Ut = 331380.3/270.54 =  1225  unidades.
En pesos:
Vt = Ut x P = 1225 x 331.46 = 406038.5
Para el trimestre 3, el punto de equilibrio se encuentra en 1225 unidades o $406,038.50 pesos.
