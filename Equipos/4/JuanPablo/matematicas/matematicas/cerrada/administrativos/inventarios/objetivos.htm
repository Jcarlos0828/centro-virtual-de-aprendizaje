Control de Inventarios
Objetivos

1.- Conocer los elementos o variables que intervienen en el control de inventarios.
2.- Construir el modelo matemático para los distintos problemas del control de inventarios.
3.- Establecer la representación algebraica del modelo en ecuaciones de diferencia finita.
4.- Utilizar la hoja de Excel para plantear y resolver problemas para el control de inventarios.
5.- Hacer una estimación del costo de pedir, cuando se conoce el costo del pedido, el consumo anual y el volumen a pedir.
6.- Hacer una estimación del costo por tener inventario, cuando se conoce el volumen a pedir y el costo por tener inventario.
7.- Hacer una estimación del Costo Total, cuando se conoce el costo por pedir y el costo por tener inventario.
8.- Hacer una estimación del LEO (lote económico óptimo) y el tiempo por pedir.
