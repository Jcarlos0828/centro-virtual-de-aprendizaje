Costo por tener Inventario

Caso 3
Estimar el costo por tener inventario para la empresa Numek
Solución para 3 meses utilizando el modelo con los elementos

Numek es una industria manufacturera dedicada a la fundición de cabezas y monoblock de aluminio para motor. Esta empresa tiene un volumen de compra por pedido de 1700 unidades al mes, y se sabe que va a consumir 20,400 al año. La empresa tiene un costo anual de inventario de $200 por unidad al año. ¿Cuál será el costo anual por tener inventario?

Para el primer mes:
De acuerdo al modelo matemático el volúmen de compra (VCE=1,700 unidades mensuales) se divide entre 2 para encontrar el inventario promedio y se multiplica por el costo por tener inventario (CPI=$200 por unidad) y con esto calculamos el Costo total por tener inventario (CTI) en este caso el costo total es (1,700/2)*$200=$170,000 para el primer mes

Para el segundo mes:
(VCE=1,700 unidades mensuales) se divide entre 2 para encontrar el inventario promedio y se multiplica por el costo por tener inventario (CPI=$200 por unidad) y con esto calculamos el Costo total por tener inventario (CTI) en este caso el costo total es (1,700/2)*$200=$170,000 más el costo total del mes anterior, entonces serían $170,000 + $170,000=$340,000 el acumulado para el segundo mes.

Para el tercer mes:
(VCE=1,700 unidades mensuales) se divide entre 2 para encontrar el inventario promedio y se multiplica por el costo por tener inventario (CPI=$200 por unidad) y con esto calculamos el Costo total por tener inventario (CTI) en este caso el costo total es (1,700/2)*$200=$170,000 más el costo total del mes anterior, entonces serían $340,000 + $170,000=$510,000 el acumulado para el tercer mes.
