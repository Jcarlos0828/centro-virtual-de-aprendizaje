Enero
CT= CPL*(CAU/VCE) + (VCE/2)*CPI*(TPI)
CT= 350*(50,000/150) + (150/2)* 250*0.25= 121,354.16

Febrero
CT= CPL*(CAU/VCE) + (VCE/2)*CPI*(TPI)
CT= 350*(50,000/1000) + (1000/2)* 250*0.25=48,750

Marzo
CT= CPL*(CAU/VCE) + (VCE/2)*CPI*(TPI)
CT= 300*(50,000/4800) + (4800/2)* 200*0.5=243,125

Abril
CT= CPL*(CAU/VCE) + (VCE/2)*CPI*(TPI)
CT= 300*(50,000/9900) + (9900/2)* 200*0.5=496,515.15

Mayo
CT= CPL*(CAU/VCE) + (VCE/2)*CPI*(TPI)
CT= 250*(50,000/15001) + (15001/2)* 150*0.75=844,639.52

Junio
CT= CPL*(CAU/VCE) + (VCE/2)*CPI*(TPI)
CT= 250*(50,000/19149) + (19149/2)* 150*0.75=1,077,784.02

TOTAL CT= 121,354.16 + 48,750 + 243,125 + 496,515.15 + 844,639.52 + 1,077,784.02= 2,832,167.85
