Estimar el lote económico óptimo para cierta compañía
Solución utilizando la fórmula

El costo del pedido es de $30 y el costo por tener inventario es el 40% del precio unitario de compra anual.
Determinar LEO (lote económico optimo) y el costo total anual.

El primer paso es encontrar el LEO para cada precio empezando con el precio menor hasta encontrar un LEO factible

LEO0.80 = (2*CAU*CPE/CUI)1/2 = (2(4000)30/.32)1/2 = 866 interruptores

Como el LEO de 866 interruptores es menor al mínimo del rango para un costo unitario de $0.80 continuaremos con el siguiente precio unitario de $0.85

LEO0.85 = (2*CAU*CPE/CUI)1/2 = (2(4000)30/.34)1/2 = 840 interruptores

Este LEO de 840 interruptores es factible ya que cae dentro del rango del precio unitario de $0.85 (500 a 999).

Ahora se calculará el costo total para 840 interruptores y se comparará con el costo total de la mínima cantidad necesaria para obtener un precio de $0.80 por interruptor.

CT = Costo por pedido + Costo por inventario 
CT = CPL*(CAU/VCE) + (VCE/2)*CPI
CT840 = (840/2)(.34) + (4000/840)30 + .85(4000) =$3,686
CT840 = (1000/2)(.32) + (4000/1000)30 + .80(4000) =$3,480
El costo mínimo se obtiene ordenando 1000 interruptores
