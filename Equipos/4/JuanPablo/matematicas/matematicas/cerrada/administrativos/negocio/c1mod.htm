Primero se debe calcular el costo total que es la suma de los costos fijos más los costos variables:
CTt = CF + CV(Ut)
Costos fijos para el trimestre 1
Los costos fijos se encuentran sumando todos los costos fijos de la tabla:
CF = Depreciación Maq ($160,000) + Mano de Obra ($60,000) + Electricidad ($56,000) + Sueldos ($100,000) + Gas ($15,000) + Teléfono ($5,000) + Papelería ($1,500) + Agua ($1,800) + Equipo de oficina ($1,750) + Publicidad ($1,000)
CF = $160,000 + $60,000 + $56,000 + $100,000 + $15,000 + $5,000 + $1,500 + $1,800 + $1,750 + $1,000
CF = $402,050
Los costos fijos para el trimestre 1 son iguales a $402,050 pesos.

Costos variables para el trimestre 1
Los costos variables unitarios se calculan a partir de la suma de todos los costos variables unitarios presentados en la tabla:
CVU = Materia Prima ($360) + Depreciación Maq. ($5.33) + Mano de Obra ($20) + Electricidad ($5.33) + Teléfono ($1.18) + Equipo de Oficina ($1.00) + Publicidad ($5.33)
CVU = $360 + $5.33 + $20 + $5.33 + $1.18 + $1.0 + $5.33
CVU = $398.17
Los costos variables se calculan multiplicando los costos variables unitarios (398.17) por las unidades vendidas (750):
CV = $398.17*(Ut)
CV = $398.17*(750)
CV = $298,627.5
Los costos variables para el trimestre 1 son iguales a $298,627.5 pesos.

  Costos totales para el trimestre 1
Los costos totales son iguales a la suma de los costos fijos (402,050) más los costos variables (298,627.5):
CT = CF + CV
CT = $402,050 + $298,627.5
CT = $700,677.5
Los costos totales para el trimestre 1 son iguales a $700,677.5 pesos.

  Ventas para el trimestre 1
Hay que calcular las ventas que sería las unidades vendidas (750) por el precio (1800):
Vt = Ut * P
Vt = 750*1,800
Vt = $1,350,000
Las ventas para el trimestre 1 son iguales a $1,350,000 pesos.

  Utilidades para el trimestre 1
Las utilidades, que serían las ventas menos los costos totales:
Ut = Vt – CTt
Ut = $1,350,000 - $700,677.5
Ut = $649,322.5
Las utilidades para el trimestre 1 son iguales a $649,322.5 pesos.

  Punto de equilibrio para el trimestre 1
El punto de equilibrio para la ciudad de Monterrey y la fórmula para encontrar el punto de equilibrio en unidades es:
Ut = CF/(P-CV)
Ut = $402,050/($1,800-$398.17)
Ut = 286.80 unidades
El punto de equilibrio se encuentra en 286.80 unidades.
