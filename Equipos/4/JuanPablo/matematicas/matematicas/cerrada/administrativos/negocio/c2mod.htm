Consumo anual (CAU): 450 unidades para el primer trimestre
Costo por Pedir(CPL): $500
Costo por tener inventario (CPI): $1.5

Para calcular los costos es necesario primero calcular el costo unitario del inventario para poder sumarlo a los costos variables.

Costo de Inventario:
Primero tenemos que encontrar el lote económico óptimo, donde los costos totales son menores, para esto utilizamos la fórmula de LEO:
LEO = ((2*CPL*CAU)/(CPI))^(1/2)
LEO = ((2*$500*450)/($1.5))^(1/2)
LEO = 547.72 redondeamos a 548 unidades.
El lote económico óptimo para el trimestre 1 es de 548 unidades.

En nuestro caso, el lote económico óptimo (LEO)lo vamos a tomar como el volúmen de compra óptimo y con esto sacar el costo total de  inventario, que es la suma del costo por pedir más el costo por tener inventario.
CI = CPE + CTI

donde,

Primero calculamos el costo por pedir:
CPE = CPL [CAU/VCE]
CPE = $500[450/548]
CPE = $410.58
Después calculamos el costo por tener inventario:
CTI = (VCE/2)*CPI
CTI = (548/2)*$1.5
CTI = $411
Ahora sumamos el Costo por pedir más el Costo por tener inventario
CI = CPE + CTI
CI = $410.58 + $411
CI = $821.58

Ahora sí, podemos calcular el costo total que es la suma de los costos fijos más los costos variables unitarios:
CT = CF + CV(Ut)

Costos Fijos:
CF = Electricidad ($10,000) + Sueldos ($50,000) + Gas ($9,000) + Teléfono ($5,000) + Papelería ($1,000) + Agua ($1,200) + Equipo de Oficina ($1,875) + Publicidad ($500) + Renta ($21,000) + Sueldos de choferes ($2,250) + Fletes ($4,000) + Bodega ($8,000)
CF = $10,000 + $50,000 + $9,000 + $5,000 + $1,000 + $1,200 + $1,875 + $500 + $21,000 + $2,250 + $4,000 + $8,000
CF = $113,825
Los costos fijos para el trimestre 1 son iguales a $113,825

Costos variables:
CV = Teléfono ($1.96) + Equipo de Oficina ($1.39) + Publicidad ($10) + Sueldos de Choferes ($1.67) +             Fletes ($2.22) + Costo unitario del inventario ($821.58)
CV = $838.82*(Ut)
CV = $838.82*(450)
CV = $377,469
Los costos variables para el trimestre 1 son iguales a $377,469


  Costos totales para el trimestre 1
Los costos totales son iguales a la suma de los Costos Fijos más los Costos Variables
CT = CF +CV
CT = $113,825 + $377,469
CT = $491,294
Los costos totales para el trimestre 1 son iguales a $491,294.

  Ventas para el trimestre 1
Hay que calcular las ventas que sería las unidades vendidas (Ut) por el precio (P) de los anillos:
Vt = (Ut)*(P)
Vt = 450*$1,800
Vt = $810,000
Las ventas para el trimestre 1 son iguales a $810,000 pesos.

  Utilidades para el trimestre 1
Después hay que calcular las utilidades (UTt) que serian las ventas (Vt) menos los costos totales (CT):
UTt = Vt-CT
UTt = $810,000 - $491,294
UTt = $318,706
Las utilidades para el trimestre 1 son iguales a $318,706 pesos.

  Punto de Equilibrio para el trimestre 1
Por último nos piden encontrar el punto de equilibrio para la ciudad de Guadalajara y la fórmula para encontrar el punto de equilibrio en unidades es:
Ut = CF/(P-CV)
Ut = $113,825/($1,800-$838.82)
Ut = 118.42 unidades
El punto de equilibrio se encuentra en 118.42 unidades.
