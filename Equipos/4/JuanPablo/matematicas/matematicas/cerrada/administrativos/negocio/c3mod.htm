Modelo operativo de un negocio.

Compañía de anillos de graduación
Regiones 1 y 2 (Monterrey y Guadalajara)
Solución para un trimestre utilizando el modelo con las fórmulas.

Se desean conocer las ventas, costos totales, utilidades y punto de equilibrio para las dos regiones en que opera la compañía Rochester, S.A. utilizando los datos de los casos 1 y 2.

Costo total: Es la suma de los Costos totales de las dos regiones
CTt = $700,677.5 + $491,294
CTt = $1,191,971.5
Los costos totales para el trimestre1 son de $1,191,971.5 pesos.

  Ventas para el trimestre 1
VTt = (750 + 450)*($1,800)
VTt = $2,160,000
Las ventas para el trimestre1 son de $2,160,000 pesos.

  Utilidades para el trimestre 1
Para las utilidades es necesario encontar las ventas de las dos regiones menos el costo total (CTt)
UTt = VTt – CTt
UTt = $2,160,000 - $1,191,971.5
UTt = $968,028.5
Las utilidades para el trimestre 1 son de $968,028.5 pesos.

   Punto de Equilibrio para el trimestre 1
El total de los Costos Fijos de las dos regiones entre el precio menos los Costos Variables unitarios.
Ut = CF/(P-CV)
Ut = ($402,050 + $113,825)/($1,800 – ($398.17+ $838.82)
Ut = 916.28 unidades
El punto de equilibrio se encuentra en 916.28 unidades.

Clasificación por tipos de Costos (Producción, Administración, Distribución):

Producción: La suma de todos los costos fijos más los costos variables unitarios
Costo total de Producción = CF + CV(Ut)
Costo total de Producción = $220,000 + $385.33*(750)
Costo total de Producción = $508,997.5

Administración: La suma de todos los costos fijos más los costos variables unitarios de las dos regiones
Costo total de Administración = CF + CV(Ut)
Costo total de Administración = $281,625 + $12.84*(750) + 13.35*(450)
Costo total de Administración = $297,262.5

Distribución: La suma de todos los costos fijos más los costos variables unitarios
Costo total de Distribución = CF + CV(Ut)
Costo total de Distribución = $14,250 + $825.47*(450)
Costo total de Distribución = $385,711.5

El total de todos es la suma de los tres: $508,997.5 + $297,262.5 + $385,711.5
El costo total es:$1,191,971.5

Dividiendo cada uno de los tipos de costo entre le total de ellos podemos saber el porcentaje de cada uno de ellos y seleccionar cual de estos tres tipos de costo nos causan más costos totales

Producción: $508,997.5 / $1,191,971.5 = 0.4270 que es el 42.70%


Administración: $297,262.5 / $1,191,971.5 = 0.2493 que es el 24.93%


Distribución: $385,711.5 / $1,191,971.5 = 0.3235 que es el 32.35%
