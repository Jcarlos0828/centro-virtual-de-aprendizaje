Utilidades para una línea de productos.

Caso 2
Determinar las utilidades de una fábrica de tostadas
Solución utilizando la formula.

La empresa Los Charritos se dedica a la fabricación de tostadas. Los tres productos que maneja son tostadas típicas, tostadas deshidratadas y tostadas con ajonjolí. Los costos fijos mensuales de la empresa son de $24000.  Se tienen los siguientes datos para el mes 1:

Tipo de tostadas Costos variables Precios Unid. vend. mes 1 % inc. mensual

1 (típicas)

 $5.75 $8.50 5500 11

2 (deshidratadas)

 $5.06 $8.50 4800 13

3 (ajonjolí)

 $6.23 $9.00 3500 5

Encuentre las utilidades para el mes 10.

Solución:

Se conoce la fórmula:
UTt = [U11(1+r1)t-1(P1-CV1) + U21(1+r2)t-1(P2-CV2) + U31(1+r3)t-1(P3-CV3)] – CF

Se escribe la fórmula para el mes 10, o t =10:
UT10 = [U11(1+r1)9(P1-CV1) + U21(1+r2)9(P2-CV2) + U31(1+r3)9(P3-CV3)] – CF

Sustituyendo los datos del problema:
UT10 = [5500(1+0.11)9(8.5-5.75) + 4800(1+0.13)9(8.5-5.06) + 3500(1+0.05)9(9-6.23)] – 24000
UT10 = [5500(1.11)9(2.75) + 4800(1.13)9(3.44) + 3500(1.05)9(2.77)] – 24000
UT10 = [5500(2.56)(2.75) + 4800(3.00)(3.44) + 3500(1.55)(2.77)] – 24000
UT10 = [(14069)(2.75) + (14419)(3.44) + (5430)(2.77)] – 24000
UT10 = [38689.75 + 49601.36 + 15041.1] – 24000
UT10 = [103332.21] – 24000
UT10 = 79332.21

Las utilidades para el mes 10 son de $79,332.21 pesos.
