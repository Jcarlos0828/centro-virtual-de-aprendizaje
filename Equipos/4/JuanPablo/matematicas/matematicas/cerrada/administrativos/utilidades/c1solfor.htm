Determinación de las utilidades en base a ventas y costos.

Caso 1
Determinar las utilidades de un puesto de tacos
Solución utilizando fórmula para obtener las utilidades

El puesto de tacos Don Manuel tiene un costo fijo mensual de $9,500 pesos y un costo variable de $4.7 por orden de tacos que vende. El precio por orden de tacos es de $15 pesos. El número de ordenes vendidas el mes de enero fue de 1200 órdenes y este número ha crecido a razón de 150 órdenes cada mes. Estime las utilidades para el mes de julio.

Datos:

U1 = 1200 órdenes
i = 150 órdenes
t = 7 meses
P = $15 pesos por orden
CV = $4.7 pesos por orden
CF = $9500 pesos
UT7 = ¿? se desean conocer las utilidades para el mes 7

Solución:

De acuerdo a la fórmula UTt = [U1 + (t-1)i] (P – CV) – CF aplicada para 7 meses:
UT7 = [U1 + (7-1)i] (P – CV) – CF
UT7 = [1200 + (6)(150)] (15 – 4.7) – 9500
UT7 = [1200 + 900] (10.3) – 9500
UT7 = 21630 – 9500
UT7 = 12,130
Las utilidades para el mes 7 son de $12,130 pesos.
