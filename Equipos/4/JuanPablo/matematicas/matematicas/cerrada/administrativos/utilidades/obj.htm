Utilidades y pérdidas
Objetivos

  1. Conocer los elementos o variables que intervienen en las utilidades.
  2. Construir el modelo matemático para obtener las utilidades.
  3. Establecer la representación algebraica del modelo en ecuaciones de diferencia finita.
  4. Utilizar la hoja de cálculo Excel para plantear y resolver problemas de utilidades.
  5. Hacer una estimación de las utilidades para un producto, una línea de productos y para el arranque de un negocio.
  6. Hacer un análisis comparativo de los datos y los resultados obtenidos en los problemas.
