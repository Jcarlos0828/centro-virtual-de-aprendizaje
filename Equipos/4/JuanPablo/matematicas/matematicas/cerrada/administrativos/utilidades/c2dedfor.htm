Deducción de la fórmula

En el Caso 1 se dedujo la fórmula para las utilidades producidas por un solo producto con crecimiento simple:
UTt = [U1 + (t-1)i] (P – CV) – CF

Para una línea de productos se puede utilizar una fórmula parecida, sólo se tienen que considerar varios aspectos. Los costos fijos siguen teniendo un valor constante y no dependen del tipo de producto. Los costos variables y los precios dependen del tipo de producto y se deben de multiplicar por el número de productos vendidos. En este caso, a diferencia del Caso 1, las unidades vendidas tienen un crecimiento compuesto, por lo que se puede escribir la siguiente ecuación que se dedujo en el caso de ventas con crecimiento compuesto:
Vt = Un(1 + r)(t-n) P
De donde se conoce que las unidades vendidas son:
Ut = Un(1 + r)(t-n)
En donde
t es el período final
n es el período inicial
r es el porcentaje de incremento.

Entonces, se puede escribir la ecuación de utilidades como:
UTt = Ut (P – CV) – CF
Pero para una línea de tres productos debe ser:
UTt = [U1t(P1-CV1) + U2t(P2-CV2) + U3t(P3-CV3)] – CF

Si se sustituye la ecuación de crecimiento compuesto para el número de unidades vendidas, cuando el período inicial n es igual a 1:

UTt = [U11(1+r1)t-1(P1-CV1) + U21(1+r2)t-1(P2-CV2) + U31(1+r3)t-1(P3-CV3)] – CF

En donde
r1, r2 y r3      son los porcentajes de incremento para cada tipo de tostada
P1, P2 y P3     son los precios para cada tipo de tostada
CV1, CV2 y CV3 son los costos variables de cada tipo de tostada.
