Caso 1

Valor futuro de una anualidad a 4 años con una tasa de interés compuesto anualizada capitalizable cada año.

¿Cuál es el valor futuro de una anualidad de $15,000.00 pesos por año pagaderos al final de cada año, durante 4 años a un interés compuesto de 8% capitalizable cada año?

Solución utilizando el modelo matemático

¿Cuál es el valor futuro de una anualidad de $15,000.00 pesos por año pagaderos al final de cada año, durante 4 años a un interés compuesto de 8% capitalizable cada año?

En anualidades el saldo de cada año va a ser igual al saldo del año anterior más los intereses del año más el pago que se efectúa cada año.

Primer año.

En el primer año el saldo inicial es de 0, así que no hay intereses durante el primer año, cuando finaliza el año se hace el primer pago de $15,000.00:

saldo inicial = 0.00

intereses primer año = 0.00

pago primer año = 15,000.00

El saldo para el primer año va a ser igual al saldo anterior más los intereses más el pago del primer año:

saldo primer año = 0.00 + 0.00 + 15,000.00

saldo primer año = 15,000.00

El saldo del primer año va a ser de $15,000.00 pesos.

Segundo año.

Para el segundo año el saldo va a ser igual al saldo anterior (15,000.00) más los intereses sobre ese saldo (15,000.00 x 0.08) más el pago (15,000.00):

saldo anterior = 15,000.00

intereses segundo año = 15,000.00 * 0.08

intereses segundo año = 1,200.00

pago segundo año = 15,000.00

saldo segundo año = saldo anterior + intereses del segundo año + pago segundo año

saldo segundo año = 15,000.00 + 1,200.00 + 15,000.00

saldo segundo año = 31,200.00

El saldo para el segundo año va a ser de $31,200.00 pesos.

Tercer año.

Para el tercer año el saldo va a ser igual al saldo anterior (31,200.00) más los intereses sobre ese saldo (31,200.00 x 0.08) más el pago (15,000.00):

saldo anterior = 31,200.00

intereses tercer año = 31,200.00 * 0.08

intereses tercer año = 2,496.00

pago tercer año = 15,000.00

saldo tercer año = saldo anterior + intereses tercer año + pago tercer año

saldo tercer año = 31,200.00 + 2,496.00 + 15,000.00

saldo tercer año = 48,696.00

El saldo para el tercer año va a ser de $48,696.00 pesos.

Cuarto año.

Para el cuarto año el saldo va a ser igual al saldo anterior ( 48,696.00) más los intereses sobre ese saldo ( 48,696 x 0.08) más el pago (15,000.00):

saldo anterior = 48,696.00

intereses cuarto año = 48,696 * 0.08

intereses cuarto año = 3,895.68

pago cuarto año = 15,000.00

saldo cuarto año = saldo anterior + intereses cuarto año + pago cuarto año

saldo cuarto año = 48,696 + 3,895.68 + 15,000.00

saldo cuarto año = 67,591.68

El valor futuro de la anualidad es de $67,591.68 al finalizar el cuarto año.

Elementos o variables que intervienen en Anualidades

El modelo establece que el saldo en cualquier año o período t es igual al saldo del año anterior más el interés producido durante el año sobre el saldo del año anterior mas el pago que se realiza cada período. Los intereses para cada año van a ser diferentes ya que en cada período se van capitalizando junto con el pago.

En seguida se definen los elementos principales en el manejo de anualidades:

S0 capital o saldo inicial.

i tasa de interés del período en su forma decimal.

t número de años o períodos.

P pago que se realiza cada período.

St saldo después de un t períodos de tiempo.

Representación algebraica del modelo de Anualidades

Para cada uno de los períodos se utiliza el mismo proceso de manera que se van a utilizar las formulas para encontrar un modelo matematico para este tipo de problemas.

En el modelo matemático para interés compuesto, el saldo para cada período va a ser igual al saldo del año anterior más el interés del saldo anterior generado durante el período más el pago.

St = St-1 + interés del período + pago del periodo

El interés para cada período va a ser el producto de multiplicar el saldo anterior (St-1) por la tasa de interés expresada en decimales ( i ).

interés del período = St-1 * i

Sustituyendo el interés y el pago P en la fórmula para el saldo se tiene

St = St-1 + St-1 * i + P

El saldo después del primer período (S1) es la suma del capital (S0) con los intereses del período (S0 * i).

S1 = S0 + S0 * i + P sacando S0 como factor común

S1 = S0 (1 + i) + P

El saldo después de un período es ---------------------------- S1 = S0 (1 + i) + P

Para obtener el saldo del segundo período (S2) se suma el saldo del primer período (S1) con el interés del período (S1 * i) más el pago del segundo período.

S2 = S1 + S1 * i + P sacando S1 como factor común

S2 = S1 (1 + i) + P

El saldo después de dos períodos es ---------------------------- S2 = S1 (1 + i) + P

Para obtener el saldo del tercer período (S3) se suma el saldo del segundo período (S2) con el interés del período (S2 * i) más el pago del tercer período.

S3 = S2 + S2 * i + P sacando S2 como factor común

S3 = S2 (1 + i) + P

El saldo después de tres períodos es ---------------------------- S3 = S2 (1 + i) + P

Al observar la fórmula para cada período se puede generalizar de la siguiente manera:

Sn = Sn-1 + Sn-1 * i + P

En el manejo de anualidades, el saldo al término de t años o periodos (St) es:

St = St-1 + St-1 * i + P

en donde,

St-1 es el saldo del período anterior. S0 viene a ser el capital o saldo inicial.

i es la tasa de interés expresada en decimales.

t es el número de años o períodos.

St es el saldo después de t períodos.

P es el pago que se realiza cada período.

Solución utilizando la representación algebraica del modelo

¿Cuál es el valor futuro de una anualidad de $15,000.00 pesos por año pagaderos al final de cada año, durante 4 años a un interés compuesto de 8% capitalizable cada año?

De acuerdo al modelo matemático, el saldo o adeudo después de cierto año t (St) , es igual al capital o préstamo del período anterior (St-1) más el interés del año más el pago que se realiza cada período (P). El interés producido en el año es el producto de multiplicar el saldo anterior (St-1) por la tasa de interés expresada en decimales.

La representación algebraica del modelo matemático para este caso sería entonces:

St = St-1 + St-1 * i + P

Primer año.

En el primer año el saldo inicial es de 0, así que no hay intereses durante el primer año, cuando finaliza el año se hace el primer pago de $15,000.00:

saldo inicial = S0 = 0.00

intereses primer año = S0 * i = 0.00 * 0.08 = 0.00

pago primer año = P = 15,000.00

El saldo para el primer año va a ser igual al saldo anterior más los intereses más el pago del primer año:

S1 = S0 + S0 * i + P

S1 = 0.00 + + 0.00 + 15,000.00

S1 = 15,000.00

El saldo del primer año va a ser de $15,000.00 pesos.

Segundo año.

Para el segundo año el saldo va a ser igual al saldo anterior (S1) más los intereses sobre ese saldo (S1 * i ) más el pago (P):

saldo anterior = S1 = 15,000.00

intereses segundo año = S1 * i = 15,000.00 * 0.08

intereses segundo año = 1,200.00

P = 15,000.00

S2 = S1 + S1 * i + P

S2 = 15,000.00 + 1,200.00 + 15,000.00

S2 = 31,200.00

El saldo para el segundo año va a ser de $31,200.00 pesos.

Tercer año.

Para el tercer año el saldo va a ser igual al saldo anterior (S2) más los intereses sobre ese saldo (S2 * i) más el pago (P):

saldo anterior = S2 = 31,200.00

intereses tercer año = S2 * i = 31,200.00 * 0.08

intereses tercer año = 2,496.00

P = 15,000.00

S3 = S2 + S2 * i + P

S3 = 31,200.00 + 2,496.00 + 15,000.00

S3 = 48,696.00

El saldo para el tercer año va a ser de $48,696.00 pesos.

Cuarto año.

Para el cuarto año el saldo va a ser igual al saldo anterior (S3) más los intereses sobre ese saldo (S3 * i) más el pago (15,000.00):

saldo anterior = S3 = 48,696.00

intereses cuarto año = S3 * i = 48,696 * 0.08

intereses cuarto año = 3,895.68

P = 15,000.00

S4 = S3 + S3 * i + P

S4 = 48,696 + 3,895.68 + 15,000.00

S4 = 67,591.68

El valor futuro de la anualidad es de $67,591.68 al finalizar el cuarto año.
