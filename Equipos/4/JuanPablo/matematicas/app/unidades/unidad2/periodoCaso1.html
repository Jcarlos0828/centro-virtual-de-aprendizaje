Interés compuesto con pago en cada período

Caso 1

Tiempo que se necesita para ahorrar cierta cantidad de dinero.

La Sra. Dávila necesita juntar $90,000.00 para cambiar su carro. Tiene ahorrado $30,000.00 y puede ahorrar $5,000.00 mensuales en una inversión que da una tasa de interés del 15% anual capitalizable cada mes.

¿Cuánto tiempo necesita para reunir la cantidad deseada?

Práctica de autoevaluación 1

  1. En la parte superior izquierda de la Tabla Interactiva se tienen los datos del Caso 1: saldo inicial (30,000), período inicial (0), período final (5), tasa mensual (1.25) y pago fijo (5,000). Cambie el período final hasta que el saldo sea mayor o igual a $90,000.00. ¿Cuántos meses se necesitan para reunir los $90,000.00?. (El número de período es 11. Se necesitan 11 meses para reunir la cantidad deseada)
  2. Si la Sra. Dávila puede ahorrar $6,000.00 mensuales, ¿cuántos meses necesitaría para ahorrar los $90,000.00?. (En 9 meses tendría $90,329.00 pesos)
  3. Seleccione el botón de Valores Iniciales. (Se vuelven a tener los valores iniciales del caso)
  4. Cambie la tasa de interés a 22% anual, dando una tasa del 1.8333% mensual, ¿cuántos meses se necesitarían para ahorrar los $90,000.00?. (En 10 meses tendría $90,309.94 pesos)
  5. Cambie la tasa de interés a 18% anual, dando una tasa del 1.5% mensual y cambie el pago fijo a $5,500.00 pesos, ¿cuántos meses se necesitarían para ahorrar los $90,000.00?. (En 9 meses tendría $90,701.76 pesos)
  6. Seleccione el botón de Valores Iniciales. (Se vuelven a tener los valores iniciales del caso)
  7. Si la Sra. Dávila decide comprar otro carro y necesita tener $130,000.00, ¿cuántos meses necesitaría para ahorrarlos?. (En 17 meses tendría $131,109.41 pesos.)

Solución utilizando el modelo matemático

La Sra. Dávila necesita juntar $90,000.00 para cambiar su carro. Tiene ahorrado $30,000.00 y puede ahorrar $5,000.00 mensuales en una inversión que da una tasa de interés del 15% anual capitalizable cada mes.

¿Cuánto tiempo necesita para reunir la cantidad deseada?

En esta unidad se utiliza el modelo de interés compuesto, la única diferencia es que cada período se va a efectuar un pago que se tiene que agregar al saldo. El saldo para cada período va a ser igual al saldo del período anterior más los intereses del período más el pago que se efectua en dicho período. Se va a ir calculando el saldo para cada mes hasta que el saldo sea mayor o igual a $90,000.00 pesos.

Primer mes.

El saldo para el primer mes va a ser igual al saldo anterior $30,000.00 más los intereses del mes más el pago que se efectua cada período.

interés del mes = 30,000.00 * 0.0125

interés del mes = 375.00 pesos

El total para el siguiente mes es de $30,000.00 más $375.00 de interés más los $5,000.00 que va a ahorrar la Sra.

saldo para el primer mes = 30,000.00 + 375.00 + 5,000.00

saldo para el primer mes = 35,375.00

El saldo después de un mes va a ser de $35,375.00 pesos.

Solución utilizando la representación algebraica del modelo en Excel

La Sra. Dávila necesita juntar $90,000.00 para cambiar su carro. Tiene ahorrado $30,000.00 y puede ahorrar $5,000.00 mensuales en una inversión que da una tasa de interés del 15% anual capitalizable cada mes.

¿Cuánto tiempo necesita para reunir la cantidad deseada?

Para utilizar los elementos matemáticos que se definieron, se tiene que incluir el ahorro mensual que se va a estar haciendo, de manera que a la representación algebraica de interés compuesto St = St-1 + St-1 * i se tiene que agregar lo que la Sra. Dávila va a ahorrar cada mes, que viene a ser el pago mensual (P) que está haciendo a su inversión, quedando la representación algebraicade la siguiente forma:

St = St-1 + St-1 * i + P

Utilizando Excel se puede hacer una tabla utilizando la representación algebraica del modelo para facilmente encontrar el número de períodos que necesita la Sra. Dávila para ahorrar los $90,000.00:

Las acciones que se tomaron para generar esta tabla son las siguientes:

En la celda B1 teclear 15%

En la celda B2 teclear =B1/12

En la celda D4 teclear 30000

En la celda B5 teclear =D4 * $B$2 y llenar hacia abajo (Fill Down)

En la celda C5 teclear 5000 y llenar hacia abajo (Fill Down)

En la celda D5 teclear =D4 + B5 + C5 y llenar hacia abajo (Fill Down)

Para ver cuantos períodos se necesitan para tener los $90,000.00, se va haciendo Fill Down hasta que el saldo sea mayor o igual a dicha cantidad, en este caso se requieren 11 períodos para tener la cantidad que se desea. Una ventaja de utilizar Excel es que si la Sra. Dávila encuentra otro banco donde la tasa sea mejor, solo tiene que modificar la celda con la tasa de interés anual y buscar en la tabla para encontrar el número de períodos que se necesitan para obtener la cantidad deseada.

El número de meses que la Sra. Dávila necesita para ahorrar $90,000.00 son 11 y su saldo sería de $92,962.41 pesos.
