Supóngase que en un estado de los Estados Unidos de América la población está integrada por sajones,latinos, y afroamericanos, y que cada grupo tiene un crecimiento diferente. Es claro que los porcentajes relativos de la población en cada uno de estos grupos va a cambiar a través del tiempo.

Hacer una estimación del número de elementos de cada uno de los grupos para el año 2020, si se tienen los siguientes datos:

En el año de 1995 la población está distribuida de la siguiente manera:

SAJONES 2,750,000

LATINOS 1,940,000

AFROAMERICANOS 1,680,000

TOTAL 6,370,000

Las tasas de crecimiento de los grupos son:

SAJONES 0.5%

LATINOS 3.2%

AFROAMERICANOS 2.7%

Cada uno de los grupos sigue el comportamiento de un modelo de Crecimiento Poblacional Compuesto. La representación algebraica del modelo, de acuerdo a lo visto en el tema de Crecimiento Poblacional Compuesto, es:

Pt = Pt-1 + Pt-1*r o bien,

Pt = Pt-1( 1 + r )

Utilizando la relación anterior se puede encontrar, para cada uno de los grupos, una representación algebraica del modelo.

Para el grupo sajón, que tiene una tasa de crecimiento de 0.5% la representación algebraica sería:

St = St-1 + St-1* ( 0.005 ) o bien,

St = St-1 ( 1 + 0.005 )

St = St-1 (1.005 )

Para el grupo latino, que tiene una tasa de crecimiento de 3.2%, la representación algebraica es:

Lt = Lt-1 + Lt-1* ( 0.032 ) o bien,

Lt = Lt-1 ( 1 + 0.032 )

Lt = Lt-1 ( 1.032 )

Lt = Lt-1 ( 1.032 )

Para el grupo de afroamericanos, que tiene una tasa de crecimiento de 2.7% la representación algebraica sería:

At = At-1 + At-1* ( 0.027 ) o bien,

At = At-1 ( 1 + 0.027 )

At = At-1 ( 1.027 )

Las tres relaciones anteriores son ecuaciones de diferencia finita que predicen el número de integrantes en cada grupo en años subsecuentes. Así, si se desea encontrar el número de integrantes del grupo sajón después de un año (en 1996) se tiene:

St = St-1 (1.005 ) al aplicar esta representación al primer año, queda la ecuación

S1 = S0 ( 1.005 ) sustituyendo S0 por su valor ( 2,750,000 ) se tiene,

S1 = 2,750,000 (1.005)

S1 = 2,763,750

La estimación del número de integrantes del grupo sajón para 1996 es de 2,763,750

El número de latinos después de un año (1996), se puede calcular de la manera siguiente:

Lt = Lt-1 ( 1.032 ) al aplicar esta representación al primer año queda la ecuación,

L1 = L0 ( 1.032) sustituyendo L0 por 1,940,000 se tiene,

L1 = 1,940,000 (1.032 )

L1 = 2,002,080

La estimación del número de integrantes del grupo latino para 1996 es de 2,002,080

Para calcular el número de habitantes afroamericanos después de un año ( 1996 ) se tiene que:

At = At-1 ( 1.027 ) al aplicar esta representación al primer año queda la ecuación,

A1 = A0 ( 1.027 ) sustituyendo A0 por su valor ( 1,680,000 ) se tiene,

A1 = 1,680,000 ( 1.027 )

A1 = 1,725,360

La estimación del número de habitantes afroamericanos para 1996 es de 1,725,360

Comparación de los porcentajes de distribución de la población en los diferentes grupos

En este tipo de casos es muy conveniente calcular el porcentaje de distribución de la población en los diferentes grupos, en el año inicial y en el final, para posteriormente hacer una comparación entre ellos.

Año inicial ( 1995 ).

En 1995 la población del estado estaba distribuida de la siguiente manera:

SAJONES 2,750,000

LATINOS 1,940,000

AFROAMERICANOS 1,680,000

TOTAL 6,370,000

Para obtener los porcentajes de distribución de la población en los diferentes grupos, se divide el número de integrantes del grupo entre el total de la población. El resultado se multiplica por 100 para que quede expresado en porciento y no en decimales.

SAJONES 2,750,000 / 6,370,000 = 0.4317 …………43.17%

LATINOS 1,940,000 / 6,370,000 = 0.3046 …………30.46%

AFROAMERICANOS 1,680,000 / 6,370,000 = 0.2637 …………26.37%

Año final ( 2020 ).

De la solución del problema por fórmula, se pueden tomar los datos de la distribución de la población en el año 2020.

Para el año 2020 la población estará distribuida de la siguiente manera:

SAJONES 3,115,188

LATINOS 4,263,774

AFROAMERICANOS 3,270,170 el total se obtiene sumando los tres grupos

EN TOTAL SON 10,649,132

Dividiendo el número de integrantes de cada grupo entre el total de la población se obtienen los porcentajes de distribución de la población en el año 2020. Éstos son:

SAJONES 3,115,188 / 10,649,132 = 0.2925 ………… 29.25%

LATINOS 4,263,774 / 10,649,132 = 0.4004 ………… 40.04%

AFROAMERICANOS 3,270,170 / 10,649,132 = 0.3071 ………… 30.71%

Calcular la tasa de crecimiento de la población total, utilizando la fórmula de Crecimiento Poblacional Compuesto

Para encontrar la tasa de crecimiento de la población total se puede utilizar la fórmula de Crecimiento Poblacional Compuesto Pt = P0 ( 1 + r )t. En la solución anterior del caso se obtuvo la población total después de 25 años (10,659,132), y se conoce además la población total inicial ( 6,370,000 ).}

Datos:

P0 = 6,370,000

P25 = 10,659,132

t = 25 años

r =? tasa de crecimiento de la población total.

Solución:

Pt = P0( 1 + r )t aplicando la fórmula para el caso específico se tiene que,

P25 = P0( 1 + r )25 despejando el factor ( 1 + r )t queda la ecuación,

( 1 + r )25 = P25 / P0 sustituyendo P25 y P0 por sus respectivos valores se tiene,

( 1 + r )25 = 10,659,132 / 6,370,000

( 1 + r )25 = 1.6733 elevando ambos miembros de la ecuación a la 1/25 se obtiene,

1 + r = ( 1.6733 )1 / 25

1 + r = 1.0208 el 1 pasa restando y nos queda la ecuación,

r = 1.0208 – 1

r = 0.0208

La tasa de crecimiento de la población total es de 2.08% anual.
