Estimación del número de libros que se venderán en cierto año

Introducción al modelo matemático

Un libro de texto es editado anualmente. En el año 1998 se vendieron 45,000 ejemplares y sabemos que las ventas se incrementan en 450 ejemplares por año. ¿Cuántos libros se espera que se vendan en el año 2003?

Esta pregunta se puede contestar de una manera muy sencilla, simplemente sumándole a la población del año anterior la cantidad de 450 tal y como se indica en la siguiente tabla:

Año No. de libros vendidos en el año
1998 45000
1999 45450
2000 45900
2001 46350
2002 46800
2003 47250

Como se puede observar cada año la venta aumenta en 450 ejemplares, sin importar la venta del año anterior. Se espera que para el año 2003 se vendan 47, 250 libros.

Sin embargo aunque la solución se determinó de una manera muy sencilla, conviene ver el planteamiento de este problema usando el concepto de modelo de crecimiento poblacional simple, así como la representación algbraica del modelo.

Elementos que intervienen en un Crecimiento Poblacional Simple.

El modelo matemático para un crecimiento poblacional simple establece que la población en cualquier año es igual a la población del año anterior más el incremento.

Como podemos observar, los elementos que intervienen en el modelo para un crecimiento poblacional simple son los siguientes:

P0 población inicial o principal.

t número de años o períodos, a partir del año o período inicial.

Pt población después de t añós o períodos.

a incremento anual o incremento por período.

Deducción de la fórmula para encontrar la población total en determinado año, en un Crecimiento Poblacional Simple, cuando se conoce la población inicial y el aumento anual.

Podemos deducir fácilmente una fórmula para estimar la población en cualquier año t (Pt), si se tiene un Crecimiento Poblacional Simple, cuando se conoce la población principal o inicial (P0) y el aumento anual (a).

Siguiendo el modelo matemático se tiene que, la población después de un año es la población del año anterior o población inicial más el aumento.

P1 = P0 + a


La población después de dos años, es la población después de un año más el aumento

P2 = P1 + a

sustituyendo la expresión encontrada previamente para P1 quedaría

P2 = (P0 + a) + a o bien,

P2 = P0 + 2a

*La población después de dos años es ----------------------------------- P2 = P0 + 2a


Si se observa la relación anterior se puede percibir que el número 2 que interviene, es el número de años después del año de la población inicial.


La población después de tres años sería la población después de dos años más el aumento.

P3 = P2 + a sustituyendo la expresión encontrada para P2 quedaría

P3 = (P0 + 2a) + a o bien,

P3 = P0 + 3a

*La población después de tres años es ----------------------------------- P3 = P0 + 3a


El número 3 que interviene, es el número de años después del año en que se tiene la población inicial. Así, si se quiere obtener la población después de cuatro años, esta sería:

P4 = P0 + 4a

*La población después de cuatro años es ------------------------------ P4 = P0 + 4a


Se puede continuar de esta manera el proceso hasta obtener la población después de t años, que sería:

Pt = P0 + ta

*La población después de t años es ---------------------------------- Pt = P0 + ta


Podemos concluir entonces que en un modelo de Crecimiento Poblacional Simple, la población después de t años se obtiene con la siguiente ecuación:

Pt = P0 + t*a

en donde,

P0 población inicial o principal.

t número de años o períodos, a partir del año o período inicial.

Pt población después de t añós o períodos.

a incremento anual o incremento por período.

El planteamiento anterior permite calcular el valor futuro del número de elementos de una población, cuando se conoce su valor inicial, su aumento anual y el número de años a transcurrir.

Estimación del número de libros que se venderán en cierto año

Un libro de texto es editado anualmente. En el año 1998 se vendieron 45,000 ejemplares y sabemos que las ventas se incrementan en 450 ejemplares por año. ¿Cuántos libros se espera que se vendan en el año 2015?

En este caso se extendió el número de años del 2003 al 2015. Utilizando la fórmula de Crecimiento Poblacional Simple Pt = P0 + t*a para resolver este problema, se tiene:

Para calcular lo anterior, primero hay que encontrar el número de años que se está prediciendo hacia el futuro, que en este caso es 2015 – 1998 que viene a ser 17 años.

Datos:

P0 = 45,000

a = 450

t = 17 años

P17 = ? se busca el número de libros después de 17 años.

Solución:

Pt = P0 + t*a aplicando la fórmula para 17 años queda la ecuación,

P17 = P0 + (17* 450)

P17 = 45,000 + 7,650

P17 = 52,650

Solución utilizando Excel

Un libro de texto es editado anualmente. En el año 1998 se vendieron 45,000 ejemplares y sabemos que las ventas se incrementan en 450 ejemplares por año. ¿Cuántos libros se espera que se vendan en el año 2003?

El problema planteado anteriormente se puede resolver de una manera muy sencilla, ya que es posible determinar una ecuación fácilmente, que permita calcular el valor de la población en un futuro. Hay casos en donde el planteamiento del modelo no genera una ecuación sencilla de resolver, o bien situaciones en donde la complejidad del modelo es tal que ni siquiera permita obtener la ecuación final de un modelo planteado. En estos casos se tiene que recurrir a métodos computacionales para resolver el modelo que se está estudiando. Una herramienta que permite plantear modelos de manera sencilla es la denominada "hoja de cálculo". Excel es un producto de este tipo.

El planteamiento en Excel del ejemplo anterior es el siguiente:

En una celda, por ejemplo A2, se registra el incremento annual de población. Se escogen dos columnas cualquiera para registrar en una el año y en la otra el valor de la población para ese año.

Por ejemplo, en la columna B se puede registrar el año y en la C la población. De esta manera el problema anterior se puede resolver haciendo B2 igual a 1998 y C2 igual a 45,000. la celda B3 se hace igual a la celda B2 más uno. La celda C3 se hace igual a la celda C2 más la celda absoluta A2. Es importante registrar A2 como absoluta para que cuando se haga la siguiente operación las relaciones futuras hagan siempre referencia a ésta. Lo último que resta es propagar laas celdas B3 y C3 hacia abajo hasta llegar al año 2015. Haciendo esto la tabla generada en Excel es la siguiente:
