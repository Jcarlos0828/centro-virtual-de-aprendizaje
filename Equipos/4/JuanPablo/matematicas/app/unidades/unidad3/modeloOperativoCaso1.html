Establecer el modelo operativo de un negocio

Caso 1

Compañía de anillos de graduación

La compañía de anillos de graduación, Rochester, S.A., recientemente expandió sus operaciones abriendo una sucursal en Guadalajara (Región 2). La fábrica y oficinas centrales se encuentran en Monterrey (Región 1), donde se producen todos los anillos.

Un porcentaje de la producción se vende en Monterrey y otro en Guadalajara. Se desean encontrar las ventas, costos totales, utilidades y punto de equilibrio de las dos regiones.

En el caso 1 se presentan los datos para Monterrey y en el caso 2, los datos para Guadalajara.

Región 1: Monterrey

Se desean encontrar las ventas, costos totales, utilidades y punto de equilibrio para la compañía Rochester S.A. en Monterrey.

Se tienen los siguientes datos:

Solución para un trimestre utilizando el modelo matemático

Región 1: Monterrey

Se desean encontrar las ventas, costos totales, utilidades y punto de equilibrio para la compañía Rochester S.A. en Monterrey.

Se tienen los siguientes datos:

Primero se debe calcular el costo total que es la suma de los costos fijos más los costos variables:

CTt = CF + CV(Ut)

Costos fijos para el trimestre 1

Los costos fijos se encuentran sumando todos los costos fijos de la tabla:

CF = Depreciación Maq ($160,000) + Mano de Obra ($60,000) + Electricidad ($56,000) + Sueldos ($100,000) + Gas ($15,000) + Teléfono ($5,000) + Papelería ($1,500) + Agua ($1,800) + Equipo de oficina ($1,750) + Publicidad ($1,000)

CF = $160,000 + $60,000 + $56,000 + $100,000 + $15,000 + $5,000 + $1,500 + $1,800 + $1,750 + $1,000

CF = $402,050

Los costos fijos para el trimestre 1 son iguales a $402,050 pesos.

Costos variables para el trimestre 1

Los costos variables unitarios se calculan a partir de la suma de todos los costos variables unitarios presentados en la tabla:

CVU = Materia Prima ($360) + Depreciación Maq. ($5.33) + Mano de Obra ($20) + Electricidad ($5.33) + Teléfono ($1.18) + Equipo de Oficina ($1.00) + Publicidad ($5.33)

CVU = $360 + $5.33 + $20 + $5.33 + $1.18 + $1.0 + $5.33

CVU = $398.17

Los costos variables se calculan multiplicando los costos variables unitarios (398.17) por las unidades vendidas (750):

CV = $398.17*(Ut)

CV = $398.17*(750)

CV = $298,627.5

Los costos variables para el trimestre 1 son iguales a $298,627.5 pesos.

1847.gif (326 bytes) Costos totales para el trimestre 1

Los costos totales son iguales a la suma de los costos fijos (402,050) más los costos variables (298,627.5):

CT = CF + CV

CT = $402,050 + $298,627.5

CT = $700,677.5

Los costos totales para el trimestre 1 son iguales a $700,677.5 pesos.

indbul2a.gif (311 bytes) Ventas para el trimestre 1

Hay que calcular las ventas que sería las unidades vendidas (750) por el precio (1800):

Vt = Ut * P

Vt = 750*1,800

Vt = $1,350,000

Las ventas para el trimestre 1 son iguales a $1,350,000 pesos.

green.jpg (756 bytes) Utilidades para el trimestre 1

Las utilidades, que serían las ventas menos los costos totales:

Ut = Vt – CTt

Ut = $1,350,000 - $700,677.5

Ut = $649,322.5

Las utilidades para el trimestre 1 son iguales a $649,322.5 pesos.

nar.jpg (784 bytes) Punto de equilibrio para el trimestre 1

El punto de equilibrio para la ciudad de Monterrey y la fórmula para encontrar el punto de equilibrio en unidades es:

Ut = CF/(P-CV)

Ut = $402,050/($1,800-$398.17)

Ut = 286.80 unidades

El punto de equilibrio se encuentra en 286.80 unidades.

Solución para un trimestre utilizando el modelo matemático

Región 2: Guadalajara

Para calcular los costos es necesario primero calcular el costo unitario del inventario para poder sumarlo a los costos variables.

Costo de Inventario:

Primero tenemos que encontrar el lote económico óptimo, donde los costos totales son menores, para esto utilizamos la fórmula de LEO:

LEO = ((2*CPL*CAU)/(CPI))^(1/2)

LEO = ((2*$500*450)/($1.5))^(1/2)

LEO = 547.72 redondeamos a 548 unidades.

El lote económico óptimo para el trimestre 1 es de 548 unidades.

En nuestro caso, el lote económico óptimo (LEO)lo vamos a tomar como el volúmen de compra óptimo y con esto sacar el costo total de inventario, que es la suma del costo por pedir más el costo por tener inventario.

CI = CPE + CTI

donde,

Primero calculamos el costo por pedir:

CPE = CPL [CAU/VCE]

CPE = $500[450/548]

CPE = $410.58

Después calculamos el costo por tener inventario:

CTI = (VCE/2)*CPI

CTI = (548/2)*$1.5

CTI = $411

Ahora sumamos el Costo por pedir más el Costo por tener inventario

CI = CPE + CTI

CI = $410.58 + $411

CI = $821.58

Ahora sí, podemos calcular el costo total que es la suma de los costos fijos más los costos variables unitarios:

CT = CF + CV(Ut)

Costos Fijos:

CF = Electricidad ($10,000) + Sueldos ($50,000) + Gas ($9,000) + Teléfono ($5,000) + Papelería ($1,000) + Agua ($1,200) + Equipo de Oficina ($1,875) + Publicidad ($500) + Renta ($21,000) + Sueldos de choferes ($2,250) + Fletes ($4,000) + Bodega ($8,000)

CF = $10,000 + $50,000 + $9,000 + $5,000 + $1,000 + $1,200 + $1,875 + $500 + $21,000 + $2,250 + $4,000 + $8,000

CF = $113,825

Los costos fijos para el trimestre 1 son iguales a $113,825

Costos variables:

CV = Teléfono ($1.96) + Equipo de Oficina ($1.39) + Publicidad ($10) + Sueldos de Choferes ($1.67) + Fletes ($2.22) + Costo unitario del inventario ($821.58)

CV = $838.82*(Ut)

CV = $838.82*(450)

CV = $377,469

Los costos variables para el trimestre 1 son iguales a $377,469

1847.gif (326 bytes) Costos totales para el trimestre 1

Los costos totales son iguales a la suma de los Costos Fijos más los Costos Variables

CT = CF +CV

CT = $113,825 + $377,469

CT = $491,294

Los costos totales para el trimestre 1 son iguales a $491,294.

indbul2a.gif (311 bytes) Ventas para el trimestre 1

Hay que calcular las ventas que sería las unidades vendidas (Ut) por el precio (P) de los anillos:

Vt = (Ut)*(P)

Vt = 450*$1,800

Vt = $810,000

Las ventas para el trimestre 1 son iguales a $810,000 pesos.

green.jpg (756 bytes) Utilidades para el trimestre 1

Después hay que calcular las utilidades (UTt) que serian las ventas (Vt) menos los costos totales (CT):

UTt = Vt-CT

UTt = $810,000 - $491,294

UTt = $318,706

Las utilidades para el trimestre 1 son iguales a $318,706 pesos.

nar.jpg (784 bytes) Punto de Equilibrio para el trimestre 1

Por último nos piden encontrar el punto de equilibrio para la ciudad de Guadalajara y la fórmula para encontrar el punto de equilibrio en unidades es:

Ut = CF/(P-CV)

Ut = $113,825/($1,800-$838.82)

Ut = 118.42 unidades

El punto de equilibrio se encuentra en 118.42 unidades.

Compañía de anillos de graduación (Operación total)

Se desean conocer las ventas, costos totales, utilidades y punto de equilibrio para las dos regiones en que opera la compañía Rochester, S.A. utilizando los datos de los casos 1 y 2.

Solución para un trimestre utilizando el modelo matemático

Costo total: Es la suma de los Costos totales de las dos regiones

CTt = $700,677.5 + $491,294

CTt = $1,191,971.5

Los costos totales para el trimestre1 son de $1,191,971.5 pesos.

indbul2a.gif (311 bytes) Ventas para el trimestre 1

VTt = (750 + 450)*($1,800)

VTt = $2,160,000

Las ventas para el trimestre1 son de $2,160,000 pesos.

green.jpg (756 bytes) Utilidades para el trimestre 1

Para las utilidades es necesario encontar las ventas de las dos regiones menos el costo total (CTt)

UTt = VTt – CTt

UTt = $2,160,000 - $1,191,971.5

UTt = $968,028.5

Las utilidades para el trimestre 1 son de $968,028.5 pesos.

nar.jpg (784 bytes) Punto de Equilibrio para el trimestre 1

El total de los Costos Fijos de las dos regiones entre el precio menos los Costos Variables unitarios.

Ut = CF/(P-CV)

Ut = ($402,050 + $113,825)/($1,800 – ($398.17+ $838.82)

Ut = 916.28 unidades

El punto de equilibrio se encuentra en 916.28 unidades.

1847.gif (326 bytes) Clasificación por tipos de Costos (Producción, Administración, Distribución):

Producción: La suma de todos los costos fijos más los costos variables unitarios

Costo total de Producción = CF + CV(Ut)

Costo total de Producción = $220,000 + $385.33*(750)

Costo total de Producción = $508,997.5

Administración: La suma de todos los costos fijos más los costos variables unitarios de las dos regiones

Costo total de Administración = CF + CV(Ut)

Costo total de Administración = $281,625 + $12.84*(750) + 13.35*(450)

Costo total de Administración = $297,262.5

Distribución: La suma de todos los costos fijos más los costos variables unitarios

Costo total de Distribución = CF + CV(Ut)

Costo total de Distribución = $14,250 + $825.47*(450)

Costo total de Distribución = $385,711.5

El total de todos es la suma de los tres: $508,997.5 + $297,262.5 + $385,711.5

El costo total es:$1,191,971.5

Dividiendo cada uno de los tipos de costo entre le total de ellos podemos saber el porcentaje de cada uno de ellos y seleccionar cual de estos tres tipos de costo nos causan más costos totales

Producción: $508,997.5 / $1,191,971.5 = 0.4270 que es el 42.70%

Administración: $297,262.5 / $1,191,971.5 = 0.2493 que es el 24.93%

Distribución: $385,711.5 / $1,191,971.5 = 0.3235 que es el 32.35%
