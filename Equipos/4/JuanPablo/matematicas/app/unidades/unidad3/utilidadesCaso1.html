Utilidades y pérdidas

Caso 1

Determinación de utilidades en base a ventas y costos.

El puesto de tacos Don Manuel tiene un costo fijo mensual de $9,500 pesos y un costo variable de $4.7 por orden de tacos que vende. El precio por orden de tacos es de $15 pesos. El número de ordenes vendidas el mes de enero fue de 1200 órdenes y este número ha crecido a razón de 150 órdenes cada mes. Estime las utilidades para el mes de julio.

Soluciónpara 3 meses utilizando el modelo matemático con los elementos

El puesto de tacos Don Manuel tiene un costo fijo mensual de $9,500 pesos y un costo variable de $4.7 por orden de tacos que vende. El precio por orden de tacos es de $15 pesos. El número de ordenes vendidas el mes de enero fue de 1200 órdenes y este número ha crecido a razón de 150 órdenes cada mes. Estime las utilidades para el mes de julio.

Las utilidades de una empresa se obtienen como los ingresos menos los egresos. Los ingresos en este caso son iguales a las ventas totales. Los egresos son iguales a los costos totales.

De la descripción del caso podemos notar que las ventas tienen un crecimiento simple, ya que se incrementan en una cantidad de 150 órdenes cada mes. Por otro lado, los costos totales son iguales a los costos fijos más los costos variables multiplicados por el número de órdenes vendidas. En este caso se está asumiendo que los tacos se hacen a la orden, lo que significa que el número de órdenes vendidas es igual al número de órdenes producidos (o la producción, como se vio en el tema de costos).

Mes 1:

Para el mes de enero (período 1), las ventas son iguales al número de órdenes producidas y vendidas (1200) multiplicado por el precio ($15):

V1 = U1 x P

V1 = 1200 x 15

V1 = 18,000 pesos

Las ventas de enero son de $18,000 pesos.

Los costos totales son iguales a los costos fijos (9500) más los costos variables (4.7) multiplicados por el número de órdenes producidas y vendidas (1200):

CT1 = CF + (CV x U1)

CT1 = 9500 + (4.7 x 1200)

CT1 = 9500 + 5640

CT1 = 15,140 pesos

Los costos totales de enero son de $15,140 pesos.

Las utilidades son iguales a las ventas (18000) menos los costos totales (15140):

UT1 = V1 – CT1

UT1 = 18000 – 15140

UT1 = 2,860 pesos

Las utilidades para enero son de $2,860 pesos.

Mes 2:

Para el mes de febrero (período 2), las ventas son iguales al número de unidades vendidas multiplicado por el precio. El número de unidades vendidas tiene un crecimiento simple. Utilizando la representación algebraica de un crecimiento simple, Ut = Ut-1+ i, las unidades vendidas en el período 2 son iguales a las unidades vendidas el período anterior (1200) más el incremento mensual de unidades (150):

U2 = U1 + i

U2 = 1200 + 150

U2 = 1350

Entonces las ventas son iguales al número de unidades vendidas (1350) multiplicado por el

precio(15):

Vt = Ut x P

V2 = (1350) (15)

V2 = 20,250 pesos

Las ventas del mes de febrero son de $20,250 pesos.

Los costos totales son iguales a los costos fijos (9500) más los costos variables (4.7) multiplicados por el número de órdenes producidas y vendidas (1350):

CT2 = CF + (CV x U2)

CT2 = 9500 + (4.7 x 1350)

CT2 = 9500 + 6345

CT2 = 15,845 pesos

Los costos del mes de febrero son de $15,845 pesos.

Las utilidades son iguales a las ventas (20250) menos los costos totales (15845):

UT2 = V2 – CT2

UT2 = 20250 – 15845

UT2 = 4,405 pesos

Las utilidades para febrero son de $4,405 pesos.

Mes 3:

Para el mes de marzo (período 3), las unidades vendidas son iguales a las unidades vendidas el período anterior (1350) más el incremento mensual de unidades (150):

U3 = U2 + i

U3 = 1350 + 150

U3 = 1500

Entonces las ventas son iguales al número de unidades vendidas (1500) multiplicado por el precio(15):

V3 = U3 x P

V3 = (1500) (15)

V3 = 22,500 pesos

Las ventas para marzo son de $22,500 pesos.

Los costos totales son iguales a los costos fijos (9500) más los costos variables (4.7) multiplicados por el número de órdenes producidas y vendidas (1500):

CT3 = CF + (CV x U3)

CT3 = 9500 + (4.7 x 1500)

CT3 = 9500 + 7050

CT3 = 16,550 pesos

Los costos totales para marzo son de $16,550 pesos.

Las utilidades son iguales a las ventas (22500) menos los costos totales (16550):

UT3 = V3 – CT3

UT3 = 22500 – 16550

UT3 = 5,950 pesos

Las utilidades para marzo son de $5,950 pesos.

Representación algebraica del modelo

De acuerdo al modelo matemático, las utilidades son iguales a los ingresos menos los egresos. En este caso, los ingresos son iguales a las ventas totales, y los egresos son iguales a los costos totales. Entonces las utilidades son iguales a las ventas totales menos los costos totales:

UTt = Vt - CTt

Las ventas son iguales al número de unidades multiplicado por el precio:

Vt = Ut x P

Los costos totales son iguales a los costos fijos más el producto de los costos variables unitarios multiplicados por el número de unidades producidas o vendidas:

CTt = CF + (CV x Ut)

El número de unidades tiene un crecimiento simple por lo que las unidades vendidas en cierto período (t) es igual a las unidades vendidas en el período anterior (t-1) más el incremento en unidades (i):

Ut = Ut-1 + i

El modelo matemático está dado por el sistema de ecuaciones:

UTt = Vt -CTt

Vt = Ut x P

CTt = CF + (CV x Ut)

Ut = Ut-1 + i

Deducción de fórmula para encontrar utilidades

Para la deducción de la fórmula se utilizarán las ecuaciones de la representación algebraica:

UTt = Vt - CTt

Vt = Ut x P

CTt = CF + (CV x Ut)

Ut = Ut-1 + i

Se pueden sustituir las fórmulas para ventas totales y costos totales en la de utilidades para dar:

UTt = Vt – CTt

UTt = Ut * P – [CF + (CV * Ut)]

Distribuyendo el signo negativo y reordenando los términos en la ecuación:

UTt = P * Ut – (CV * Ut) – CF

Factorizando Ut como factor común:

UTt = Ut (P – CV) – CF

Mes 1

Para el mes 1, el número de unidades vendidas es un dato:

Ut =U1

Mes 2

Para el mes 2, el número de unidades vendidas es igual al número de unidades vendidas el mes anterior (U1) más el incremento (i):

U2 = U1 + i

Mes 3

Para el mes 3, el número de unidades vendidas es igual al número de unidades vendidas el mes anterior (U2) más el incremento (i):

U3 = U2 + i

Pero, U2 = U1 + i, así que sustituyendo esta ecuación en la anterior se tiene que:

U3 = U1 + 2i

Del comportamiento de las unidades vendidas se puede notar que el coeficiente del incremento (i) es uno menos que el número del período en el que se evalúa el número de unidades, por lo que se puede escribir que para un período t, cuando se conocen las utilidades en un período n es:

Ut = Un + (t-n)i

Sustituyendo la ecuación del número de unidades en la ecuación de utilidades se obtiene:

UTt = [Un + (t-n)i](P – CV) – CF

Se puede concluir que la fórmula para encontrar las utilidades para un período t (UTt) cuando el número de unidades producidas y vendidas tiene un crecimiento simple es:

UTt = [Un + (t-n)i](P – CV) – CF

en donde,

Un es el número de unidades vendidas en el período n (dato inicial).

n es el período inicial.

t es el período en que se desean evaluar las utilidades.

i es el incremento de unidades vendidas, por período.

P es el precio por unidad.

CV costos variables unitarios

CF costos fijos

Fórmula para encontrar las utilidades

Solución utilizando fórmula para obtener las utilidades

El puesto de tacos Don Manuel tiene un costo fijo mensual de $9,500 pesos y un costo variable de $4.7 por orden de tacos que vende. El precio por orden de tacos es de $15 pesos. El número de ordenes vendidas el mes de enero fue de 1200 órdenes y este número ha crecido a razón de 150 órdenes cada mes. Estime las utilidades para el mes de julio.

Datos:

U1 = 1200 órdenes

i = 150 órdenes

t = 7 meses

P = $15 pesos por orden

CV = $4.7 pesos por orden

CF = $9500 pesos

UT7 = ¿? se desean conocer las utilidades para el mes 7

Solución:

De acuerdo a la fórmula UTt = [U1 + (t-1)i] (P – CV) – CF aplicada para 7 meses:

UT7 = [U1 + (7-1)i] (P – CV) – CF

UT7 = [1200 + (6)(150)] (15 – 4.7) – 9500

UT7 = [1200 + 900] (10.3) – 9500

UT7 = 21630 – 9500

UT7 = 12,130

Las utilidades para el mes 7 son de $12,130 pesos.
