Punto de equilibrio

Caso 1

Punto de equilibrio para una fábrica de pays.

PieGrande, una compañía que fabrica pays de manzana, fija el precio de cada pay en $60 pesos. Los costos fijos mensuales de la empresa son iguales a $40,000 pesos. El costo variable unitario (por pay) es de $20 pesos. Determine la cantidad de pays que se deben producir y vender para no tener ganancias o pérdidas (el punto de equilibrio). ¿Cuáles son los costos totales de producir esa cantidad de pays?

Solución utilizando el modelo matemático con los elementos.

  1. En la parte superior izquierda de la Tabla Interactiva, fueron vaciados algunos datos del Caso 1: costo fijo (40000), costos variables (20) y precio (60).
  2. Para encontrar la solución de este caso se supuso que en el mes 0 no se tenían unidades producidas y se manejó un incremento mensual de 100 unidades, de manera que para obtener el número de unidades en un mes basta con multiplicar el número del mes por 100. Observe que se alimentaron estos valores en los espacios de unidades iniciales (0), crecimiento mensual (100) mes inicial (1) y mes final (5). Note que la tabla interactiva no muestra la solución al problema todavía ya que se busca encontrar el punto en que las gráficas de costos y de ventas tengan el mismo valor o se intersecten. Este punto es el mismo en el que la gráfica de las utilidades cruce el valor de cero.
  3. Por tanteo, mueva el valor del mes final hasta que se crucen las gráficas de costos y ventas. Observe el valor que tienen las unidades en la tabla de resultados correspondiente al mes en que se intersectan las gráficas. A este valor en las unidades se le llama punto de equilibrio.
  4. Ahora suponga que el precio se tuvo que reducir a $40. Alimente este nuevo valor y observe los cambios. Por tanteo, mueva el valor del mes final hasta que encuentre el nuevo punto de equilibrio.
  5. Oprima el botón de valores iniciales. Cambie el valor del precio a $80 y encuentre el nuevo punto de equilibrio. ¿Qué puede concluir a cerca del comportamiento del punto de equilibrio cuando el precio se aumenta o se disminuye?
  6. Oprima el botón de valores iniciales. Si el valor de los costos fijos se aumenta a $60,000, cambie este dato en la tabla interactiva y encuentre el punto de equilibrio por tanteo.
  7. Ahora alimente un valor de costos fijos igual a $30,000 en la tabla y encuentre el punto de equilibrio por tanteo. ¿Qué puede concluir a cerca del comportamiento del punto de equilibrio cuando los costos fijos aumentan o disminuyen?
  8. Oprima el botón de valores iniciales. Suponga que los costos variables unitarios cambian y toman un valor igual a $40. Alimente este valor y observe como la gráfica de los costos se acerca a la de ventas. Por tanteo, encuentre el punto de equilibrio.
  9. Ahora cambie el valor de los costos variables unitarios a $10. Observe como las gráficas de ventas y costos se separan, y las utilidades aumentan. Calcule el punto de equilibrio para esta situación. ¿Qué puede concluir a cerca del comportamiento del punto de equilibrio cuando los costos variables unitarios aumentan o disminuyen?
  10. Efectúe cambios en el crecimiento anual, año final, precio, costos fijos y variables e incluso en el valor de las unidades iniciales y observe los efectos que dichos cambios tienen en las utilidades y en el punto de equilibrio tanto en la tabla como en la gráfica.

Solución utilizando el modelo matemático con los elementos.

PieGrande, una compañía que fabrica pays de manzana, fija el precio de cada pay en $60 pesos. Los costos fijos mensuales de la empresa son iguales a $40,000 pesos. El costo variable unitario (por pay) es de $20 pesos. Determine la cantidad de pays que se deben producir y vender para no tener ganancias o pérdidas (el punto de equilibrio). ¿Cuáles son los costos totales de producir esa cantidad de pays?

El punto de equilibrio se encuentra al calcular el número de unidades producidas y vendidas que hacen que no se tengan ganancias o pérdidas, o sea que las utilidades tengan un valor de cero. En este caso, los ingresos (o ventas) se cancelan con los egresos (o costos). La fórmula de utilidades para un período t, establece que las utilidades (UTt) son iguales a las ventas (Vt) menos los costos totales (CTt):

UTt = Vt -CTt

Para el caso, las ventas (Vt) se pueden calcular como el número de unidades vendidas (Ut) multiplicado por el precio (60):

Vt = 60 Ut

Los costos totales (CTt) se pueden expresar como los costos fijos (40000) más el costo variable unitario (20) multiplicado por el número de unidades vendidas (Ut):

CTt = 40000 + 20 Ut

Las utilidades (UTt) son iguales a las ventas (60 Ut) menos los costos totales (40000 + 20 Ut):

UTt = 60 Ut - (40000 + 20 Ut)

Simplificando y agrupando términos comunes:

UTt = 60 Ut - 40000 - 20 Ut

UTt = 40 Ut - 40000

El punto de equilibrio se encuentra al igualar las utilidades a cero y despejar para encontrar el número de unidades:

UTt = 40 Ut - 40000 = 0

40 Ut - 40000 = 0

40 Ut = 40000

Ut = 40000/40 = 1000 unidades.

El punto de equilibrio se encuentra en 1000 unidades

De tal manera que el punto de equilibrio puede expresarse como 1000 unidades o $60000 pesos.

Representación algebraica del modelo.

El modelo matemático en este caso sería:

Las ventas para un año (Vt ) son iguales al número de unidades vendidas en ese año (Ut ) multiplicadas por el precio por unidad (P).

Vt = Ut x P

Los costos totales (CTt) son iguales a los costos fijos (CF) más el producto del costo variable unitario (CV) multiplicado por el número de unidades (Ut).

CTt = CF + (CV x Ut)

Las utilidades (UTt) se pueden expresar como las ventas (Vt) menos los costos totales (CTt).

UTt = Vt - CTt

Este sistema de tres ecuaciones

Vt = Ut x P

CTt = CF + (CV x Ut)

UTt = Vt - CTt

constituye la representación algebraica del modelo.

Deducción de fórmula para encontrar el punto de equilibrio

Para hacer la deducción de la fórmula se utilizará el sistema de ecuaciones que representan el modelo:

Vt = Ut x P

CTt = CF + (CV x Ut)

UTt = Vt - CTt

Si se sustituyen las ecuaciones de ventas y costos totales en la de utilidades se tiene:

UTt = Vt - CTt

UTt = Ut x P - [CF + (CV x Ut)]

Distribuyendo el signo negativo:

UTt = Ut x P - CF - (CV x Ut)

Agrupando los términos que tienen el número de unidades (Ut) como factor común:

UTt = [(Ut x P) - (CV x Ut)] - CF

Si se factoriza el número de unidades (Ut) de los primeros dos términos:

UTt = [Ut (P - CV)] - CF

El punto de equilibrio se encuentra cuando las utilidades son iguales a cero, entonces se iguala la ecuación anterior a cero:

UTt = [Ut (P - CV)] - CF = 0

Se despeja el número de unidades (Ut) de le ecuación:

[Ut (P - CV)] - CF = 0

Ut (P - CV) = CF

Ut = CF/(P - CV)

Entonces el punto de equilibrio para un período t y un producto dado, asumiendo que el número de unidades producidas es igual al número de unidades perdidas se puede encontrar con la fórmula:

Ut = CF/(P - CV)

donde,

Ut es el punto de equilibrio en número de unidades vendidas.

P es el precio por unidad.

CV costos variables unitarios

CF costos fijos

Una vez que se tiene el punto de equilibrio en número de unidades, se puede expresar en pesos utilizando la fórmula de ventas simplemente multiplicándolo por el precio.

Vt = Ut x P

Solución utilizando fórmula para obtener el punto de equilibrio

PieGrande, una compañía que fabrica pays de manzana, fija el precio de cada pay en $60 pesos. Los costos fijos mensuales de la empresa son iguales a $40,000 pesos. El costo variable unitario (por pay) es de $20 pesos. Determine la cantidad de pays que se deben producir y vender para no tener ganancias o pérdidas (el punto de equilibrio). ¿Cuáles son los costos totales de producir esa cantidad de pays?

Datos:

P = $60 pesos

CV = $20 pesos por pay

CF = $40000 pesos

Ut = ¿? se desea conocer el número de unidades en el punto de equilibrio

Solución:

De acuerdo a la fórmula Ut = CF/(P - CV), aplicada a los datos, se obtiene la ecuación:

Ut = CF/(P - CV)

Ut = 40000/(60 - 20)

Ut = 40000/40

Ut = 1000

El punto de equilibrio para la fábrica de pays se encuentra en 1000 unidades.

Si se desea conocer el punto de equilibrio en pesos entonces se usa la fórmula de ventas:

Vt = Ut x P

Vt = 1000 x 60

Vt = 60000

El punto de equilibrio ocurre a 60,000 pesos.
