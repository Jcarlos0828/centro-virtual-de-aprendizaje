Estimación de ventas

Caso 2

Ventas de una empresa de servicios de internet.

Se conoce que el número de usuarios de Conectix, una empresa proveedora de servicios de internet, se incrementa en un 35% mensual. La renta mensual que paga cada usuario por el servicio es de $250 pesos. Si en junio de 1999 se tenían 1346 usuarios, ¿cuáles serán los ingresos por ventas que se tendrán en diciembre de 1999?

Solución para 4 meses utilizando el modelo matemático

Se conoce que el número de usuarios de Conectix, una empresa proveedora de servicios de internet, se incrementa en un 35% mensual. La renta mensual que paga cada usuario por el servicio es de $250 pesos. Si en junio de 1999 se tenían 1346 usuarios, ¿cuáles serán los ingresos por ventas que se tendrán en diciembre de 1999?

El modelo matemático para este caso establece que las ventas en cualquier mes se pueden obtener multiplicando el número de usuarios por el pago mensual por usuario (250 pesos).

El número de usuarios en cualquier año t (Ut) es igual al número de usuarios del mes anterior (Ut-1) mas el incremento de usuarios en el mes. El incremento en el número de usuarios es el producto de multiplicar el número de usuarios del mes anterior(Ut-1) por la tasa de crecimiento expresada en decimales(r = 0.35).

Como se puede observar el incremento en el número de usuarios, guarda el comportamiento de un Crecimiento Compuesto. La representación algebraica de un modelo de Crecimiento Poblacional Compuesto es Pt = Pt-1+ Pt-1* r o bien, sacando Pt-1 de factor común Pt = Pt-1(1 + r).

La representación algebraica del modelo de Crecimiento Compuesto para encontrar el número de usuarios en cualquier período t puede ser escrita de la siguiente manera.

Ut = Ut-1+ Ut-1* r o bien,

Ut = Ut-1(1 + r).

Mes de junio.

Para encontrar las ventas del mes de junio se multiplica el número de usuarios del mes (1346) por la renta mensual que paga cada usuario (250 pesos).

Ventas de junio = 1,346 x 250

Ventas de junio = 336,500 pesos

Las ventas del mes de junio son de 336,500 pesos

Elementos o variables que intervienen en el manejo de ventas con crecimiento compuesto

El modelo de crecimiento compuesto en las ventas establece que el número de unidades vendidas, o en este caso el de usuarios de un servicio, es igual al número de usuarios del período anterior más el incremento del mes. El incremento se calcula multiplicando el número de usuarios del mes anterior por un porcentaje de incremento en decimales. Finalmente las ventas se calculan multiplicando el número de usuarios por el precio de la renta por usuario.

Se pueden observar los siguientes elementos en la descripción del modelo matemático:

n período inicial

Un número de usuarios en el período inicial n

r tasa de crecimiento en el número de usuarios por período (en decimales)

t período t

it incremento de usuarios en el período t

Ut número de usuarios en el período t

Ut-1 número de usuarios en el período t-1

P precio de renta por usuario (o por unidad)

Vt ventas en el período t

Representación algebraica del modelo compuesto de ventas

De acuerdo al modelo las ventas en cualquier período t (Vt) se calculan multiplicando el número de usuarios del período (Ut) por la renta mensual por usuario(P).

Vt = Ut x P

El número de usuarios por período, esta dado como un Crecimiento Compuesto y su representación algebraica es Ut = Ut-1(1 + r).

El sistema de ecuaciones

Vt = Ut x P t =1,2,3……

Ut = Ut-1(1 + r) t = 2,3,4,…….

determinan la representación algebraica del modelo compuesto de ventas.

Si el período inicial es n, los valores de t serían de n en adelante.

Deducción de fórmula para las ventas con crecimiento compuesto

Las ventas en cualquier período t (Vt) se calculan multiplicando el número de usuarios del período(Ut) por la renta que cada usuario paga en el período (P).

Vt = Ut x P t =1,2,3……

Las ventas en cualquier período t están dadas por……………………Vt = Ut x P

Para encontrar el número de usuarios en cualquier período t (Ut) se puede utilizar la fórmula

Pt = P0( 1 + r )t, ya que el incremento en usuarios sigue el comportamiento de un Crecimiento Compuesto.

La fórmula hay que adaptarla al caso de usuarios (unidades), al sustituir la P por U queda la ecuación

Ut = U0( 1 + r )t. El período inicial en el número de usuarios no es 0 sino 1 o n. Si el período es 1 la fórmula quedaría:

Ut = U1( 1 + r )t-1 , Si el período inicial es tomado como n , se sustituye la n en el lugar del 1 y queda la fórmula Ut = Un( 1 + r )t-n.

La fórmula Ut = Un( 1 + r )t-n es la más general ya que se puede aplicar para cualquier período inicial de 1 en adelante.

El número de usuarios en cualquier período t esta dado por …… Ut = Un( 1 + r )t-n

Se tiene que las ventas en cualquier período t se obtienen con la fórmula Vt = Ut x P en donde

Ut = Un( 1 + r )t-n, sustituyendo el valor de Ut de la segunda ecuación en la primera se llega a la expresión:

Vt = Un( 1 + r )t-n x P

Se puede concluir entonces que la fórmula para encontrar las ventas en cualquier período t cuando se tiene un Crecimiento Compuesto es:

Vt = Un( 1 + r )t-n x P

en donde,

n es el período inicial.

Un es el número de unidades (usuarios) vendidas en el período inicial n.

t es el período del que se desean saber las ventas (t > n).

r tasa de crecimiento del número de unidades (usarios) vendidas.

P precio por unidad(usuario).

Fórmula para las ventas con crecimiento compuesto

La fórmula para encontrar las ventas en cualquier período t cuando se tiene un Crecimiento Compuesto es:

Vt = Un( 1 + r )t-n x P

en donde,

n es el período inicial.

Un es el número de unidades (usuarios) vendidas en el período inicial n.

t es el período del que se desean saber las ventas (t > n).

r tasa de crecimiento del número de unidades (usarios) vendidas.

P precio por unidad(usuario).

Solución utilizando fórmula para obtener las ventas con crecimiento compuesto

Se conoce que el número de usuarios de Conectix, una empresa proveedora de servicios de internet, se incrementa en un 35% mensual. La renta mensual que paga cada usuario por el servicio es de $250 pesos. Si en junio de 1999 se tenían 1346 usuarios, ¿cuáles serán los ingresos por ventas que se tendrán en diciembre de 1999?

Datos:

U6 = 1346 usuarios

r = 0.35 incremento mensual de usuarios

P = $250 pesos mensuales por usuario

n = 6

t = 12

V12 = ¿? se desean conocer las ventas en pesos para diciembre

Solución:

De acuerdo a la fórmula Vt = Un(1 + r)(t-n) P aplicada para t =12 se tiene:

V12 = U6 (1 + r)(12 - 6) P

V12 = 1346(1 + 0.35)(6) (250)

V12 = 1346 (1.35)6 (250)

V12 = 1346 (6.05)(250)

V12 = (8148)(250)

V12 = 2,037,000

Las ventas para diciembre son de $2,037,000 dólares.
