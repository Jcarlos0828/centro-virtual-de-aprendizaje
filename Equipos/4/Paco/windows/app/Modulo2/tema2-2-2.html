Tema 2: Panel de control

Subtema 2. 2: Agregar o quitar programas

Para quitar un programa, de debe seguir los siguientes pasos:

Haga clic en cada pestaña.
  • Windows XP
  • Windows 7
  • Abrir la opción de Agregar o quitar programas en el Panel de control (control panel).
  • En la ventana que aparece, hay un listado de todos los programas existentes. Seleccionar el programa que se desea quitar.
  • Una vez seleccionado el programa, se activa debajo del nombre del programa un botón con el nombre de Cambiar/Quitar (change / quit) Hacer clic sobre ese botón.

  • Se abrirá un asistente de desinstalación y se deberán seguir las instrucciones. Dichas instrucciones hacen referencia a la seguridad de querer desinstalar el programa del sistema operativo.

Sólo los programas basados en Windows se pueden quitar mediante Propiedades de Agregar o quitar programas. Para todos los demás programas, se debe consultar la documentación correspondiente para determinar los archivos que se deben quitar.

  • Abrir el panel de control (control panel)
  • Abrir la vista a Iconos pequeños o grandes. (view by small/large icons)
  • Abrir la opción de Programas y características (Programs and features).
  • En la ventana que aparece, hay un listado de todos los programas existentes. Seleccionar el programa que se desea quitar.
  • Una vez seleccionado el programa, se activaran las opciones de Desinstalar o  Cambiar. Hacer clic sobre la opción deseada.
