Tema 2: Panel de control

Subtema 2. 2: Agregar o quitar programas

Ahora bien, para instalar un programa (install a program), se debe debe:

Haga clic en cada pestaña.
  • Windows XP
  • Windows 7
  • Abrir la opción de Agregar o quitar programas en el Panel de control (control panel).
  • Aparecerá una pantalla con el listado de todos los programas ya existentes. En la misma ventana, del lado izquierdo viene un menú, seleccionar Agregar nuevos programas.


  • En la nueva ventana que se presenta, se debe seleccionar de donde se obtendrá el programa a instalar. Si se desea de un CD o un disquete se debe hacer clic sobre el botón que dice CD
  • Si se desea obtener drivers o actualizaciones hacer clic en Windows Update. Para posteriormente bajarlos de la página de Internet de Microsoft.
  • Posteriormente de haber seleccionado la opción, aparecerá el asistente para instalación.
  • Seguir las instrucciones que este asistente va preguntando para lograr definir la ruta de instalación.

    Por último Windows manda una ventana de aviso, indicando el grado de instalación.

  • Abrir el Panel de control (Control panel)
  • Abrir la vista a Iconos pequeños o grandes (View by small/large icons)
  • Abrir la opción de Programas y caracteristicas (Programs and features)
  • Seleccionar la opción Activar/desactivar las características de Windows (Turn Windows features on or off)

  • Las características aparecen como carpetas con una casilla de verificación, y algunas con subcarpetas.
  • Para activarlas o desactivarlas, debemos marcar o desmarcar la casilla. Al marcar una opción con sub-opciones, se marcarán todas estas.
  • Si queremos desmarcar una subcarpeta, pulsaremos el pequeño icono  para desplegar el contenido de la carpeta contenedora e ir seleccionando.
  • Hacer clic en el botón de Aceptar (Ok).
